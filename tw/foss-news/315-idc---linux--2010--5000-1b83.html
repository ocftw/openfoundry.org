IDC 預測 中國 Linux 成長 2010 年達 5000 萬美元

建立日期 2006-04-22 00:00      最近更新在 2011-08-12 10:05 作者是 謝良奇

IDC 最新研究報告指出，中國 Linux 市場在未來五年內會有蓬勃成長，隨著國外 Linux 廠商持續投入中國市場，2006 年可預見的激烈競爭將帶來價格割喉戰，促使中國本地 Linux 廠商與國外廠商結盟合作。

根據 IDC 的中國 Linux 2006-2010 預測與分析 (China Linux 2006-2010 Forecast and Analysis) 報告，中國 Linux 市場從 2004 年到 2005 年成長了 27.1%，達 1180 萬美元。然而，與全球 Linux 市場的 57 億美元相較，中國 Linux 市場仍僅佔極小部份。中國 Linux 在 2005 年的成長主要來自大型政府採購，以及銀行、電信與網咖業者以 Linux 取代 SCO Unix 的平台轉換。 此外，IDC 該份研究報告預測中國 Linux 市場從 2006 到 2010 年，將有 34% 的成長，市值可達 5110 萬美元。 面對 Linux 在中國市場的進逼，SCO 以最新版 Unix 作業系統 OpenServer 6 積極穩定中國市場，SCO 全球銷售與行銷執行副總裁 Tim Negris 表示，SCO 提供中國許多大型銀行等機構技術基礎建設，一直是中國重要的技術供應商。 儘管如此，中國已有大型銀行開始嘗試採用 Linux。Turbolinux 日前與中國工商銀行簽約，將在該銀行網路中整合 Linux 系統。 IDC 的報告中顯示，2005 年 Novell 與 Red Hat 等國際 Linux 廠商透過行銷以及在 Linux 伺服器市場的成功，積極地想打入中國 Linux 市場。對於電信與金融等關鍵產業，國際大廠的確較具有競爭優勢。 雖然 2006 年的中國 Linux 將持續成長，然而 IDC 預測 2005 年帶動成長關鍵因素之一的政府採購訂單將會減少。IDC 分析師表示，2005 年中國 Linux 市場呈現的空前競爭，在 2006 年將更趨劇烈，因此價格戰將無可避免，成為中國 Linux 市場的主要障礙。 預料中國本土廠商與外國廠商將採取策略聯盟或併購，以面對激烈的競爭態勢。去年 China Standard Software 與 Novell 共組策略聯盟，日本 Turbolinux 也與中國 Co-Create 結盟。IDC 認為 2006 年一旦本土廠商過去依賴的政府採購案減少，將帶動更多這類結盟關係出現。
