CrossOver 6.0 測試版本釋出 Linux 也可以玩魔獸

建立日期 2006-09-30 08:00      最近更新在 2007-03-09 01:51 作者是 謝良奇
CodeWeavers 日前發表了 CrossOver 6.0 首次公開測試版本，給了喜歡玩 Windows 平台遊戲的 Linux 用戶一個大驚喜，新增支援魔獸世界 (World of Warcraft)、戰慄時空 2 (Half Life 2)、絕對武力 (Counterstrike) 等重量級遊戲。同時，該公司也釋出 CrossOver Mac 公開測試版本，讓 Mac 用戶在 Mac OS X 系統上暢快執行 Windows 程式。

CodeWeavers 是著名開放源碼專案 Wine 的贊助者之一，這套專案在 Unix 下重新實作 Win32 API，讓執行 Unix-based 作業系統的電腦，能夠直接執行 Windows 應用軟體。CrossOver 則是 CodeWeavers 推出的 Wine 商業版本。 CodeWeavers 執行長 Jeremy White 表示，這次釋出的仍是測試軟體，還有很多問題待解。他們非常樂意聽取測試用戶的意見。 企業 Linux 用戶可能會相當樂於聽到 CrossOver Linux 6.0.0beta2 包括了對 Outlook 2003、Flash Player 9、Visio 2003 與微軟 Project 2003 的支援。此次的測試版本還加入了 CrossOver 第 5 版以來 Wine 核心的重大修改。 此外，新的版本改善了對 Quicken 的支援。White 表示，Quicken 還有一個跟視窗位置有關的臭蟲，他們正在設法解決。 CodeWeavers 同時也釋出了 CrossOver Mac 的新公開測試版本，為希望執行 Windows 軟體的 Mac 用戶，打開了 Apple 雙重開機方案 Boot Camp 與 Parallels、VMware 等虛擬環境軟體之外的第三種可能。 透過 CrossOver Mac，Intel Mac 的使用者可以不用重新開機，也無須特別額外安裝一份 Windows 作業系統在 Mac 中，就能執行 Windows 軟體或遊戲。CrossOver Mac 支援包括微軟 Office 2003、Visio、Internet Explorer 6.0 等 50 種應用軟體與戰慄時空 (Half Life) 等遊戲。 White 建議有意親自體驗這套軟體的用戶，瀏覽該公司測試中心網站公告上的錯誤報告與解決技巧，有任何問題或報告可以通知其測試中心。
