SugarCRM 推出 FastStack 提供 Apache、PHP、MySQL 整合軟體堆疊

建立日期 2006-11-18 08:00      最近更新在 2007-03-08 08:12 作者是 謝良奇
為簡化用戶安裝與整合各類型軟體的煩瑣步驟，商用開放源碼客戶關 係管理 (CRM) 軟體供應商 SugarCRM 推出 Sugar FastStack，這套 服務提供了用戶簡單而迅速的方式，完整地安裝一整套開放源碼軟體 解決方案，包括 Sugar 的 Sugar Professional 或 Enterprise 產 品以及 Apache Web 伺服器、PHP 和 MySQL 資料庫。 Sugar FastStack 揉合了隨選 (On-Demand) 解決方案的快速部署特 性以及實地 (On-Site) 解決方案的高用戶操控性及高彈性。非技術 導向的用戶也可在短時間內安裝架設出業界首屈一指的開放源碼技術。

FastStack 會安裝 Apache Web 伺服器、PHP 與 MySQL 資料庫與 Sugar Professional 或 Sugar Enterprise。FastStack 可安裝於 SugarCRM 所支援的 Linux、Windows 與 Mac OS X 三大平台上。

建構軟體元件堆疊是目前開放源碼廠商的趨勢，如 OpenLogic、紅帽 (Red Hat)、SourceLabs 與 SpikeSource 都是如此，其目標不外乎 加速開放源碼技術的普及，將整合多項元件的複雜工作，交由廠商自 行吸收解決，而非由客戶承擔。

SugarCRM 創辦人之一的 Clint Oram 表示，他們並不想踏入軟體堆 疊業務，他們是一間商用應用軟體公司，目前正在做的是希望為他們 的用戶，簡化其應用軟體底層的技術。Oram 認為 FastStack 對於缺 少技術專業的小型企業會很有吸引力，因為盡管對於一般人要安裝個 別的開放源碼元件並不難，將其整合並予以最佳化難度則較高。

SugarCRM 將提供 FastStack 所有元件的完整支援，Oram 表示目前 該公司尚未遇到需要求助於 Apache 軟體基金會、MySQL AB 或 Zend 的問題。他補充說如果用戶有必要尋求進階協助，該公司與 MySQL 從 2004 年來一直就是合作夥伴，另外，該公司與 BitRock S 也有 合作關係，提供 Apache 與 PHP 的進階支援。

SugarCRM 預計擴充 FastStack 以支援其它主要網站伺服器如微軟 Internet Information Server，以及其 CRM 應用軟體支援的另外兩 套資料庫系統 SQL Server 與 Oracle 資料庫。FastStack 已經開始 提供給 Sugar Professional 與 Sugar Enterprise 的用戶，費用為 499 美元。
