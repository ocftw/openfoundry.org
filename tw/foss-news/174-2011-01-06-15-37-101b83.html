嵌入式系統技術工作坊

建立日期 2005-02-23 16:00      最近更新在 2011-01-06 23:37 作者是 webmaster

Linux 在嵌入式系統領域的發展前景極受曙目，但進入門檻又常讓許多人望而卻步。中央研究院資訊科學研究所 自由軟體鑄造場計畫特別邀請光寶科技公司 新技術暨產品研發中心 曾瑞源資深經理來分享他個人在這領域的專業經驗。曾瑞源資深經理早年以曾桑為名，引介 Linux 學習及應用的入門知識，如今，曾經理把累積多年的 Embedded Linux 開發及教學經驗，整理成一套 Minidoochun (迷你豆漿) 教學環境，內含最佳化的 Red Hat 9.0 Linux 系統，使用者可在Windows 及 VMware 作業環境進行練習，不會影響到既有PC內的資料。

Minidoochun 是一套嵌入式系統 rootfs 的模具組。利用這樣的模具，可以快速製作定型化的 rootfs，而且修改是非常容易的，只需在預定的rootfs 放置所需的應用程式以及函式庫，十分鐘後就可以製作出全新的 rootfs。以 Minidoochun 做為 Embedded Linux 教學環境，搭配 VMware 這套硬體平台模擬軟體，使用者可以獨立在 Windows XP 作業環境上運行及測試，快速練習各式Embedded Linux 的設定與應用程式開發；甚至搭配 skyeye 軟體可模擬多款 ARM CPU環境，練習開發 ARM 環境上的驅動程式與應用程式。
