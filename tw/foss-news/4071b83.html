Oracle 將發展 Eclipse EJB 3.0 物件關聯工具

建立日期 2005-04-08 08:00      最近更新在 2007-03-08 07:59 作者是 謝良奇
Eclipse 下的 Java 開發者有福了！Oracle 日前宣佈將發起一項旨 在推動 EJB（Enterprise JavaBeans）3.0 規格的專案，為 Eclipse 開放源碼社群提供支援 EJB 3.0 規格的物件關聯對映工具（Object- Relational Mapping）。 Oracle J2EE 應用軟體伺服器產品經理 Dennis MacNeil 表示，該專 案將支援作為 J2EE 5.0 基礎的 EJB 3.0 規格，可大幅度簡化應用 軟體開發工作，並提升開發者的生產力，此一工具將奠基於 Oracle TopLink 之上。

MacNeil 說，此套 Java 物件關聯工具及部署平台可為關鍵企業應用 軟體提供物件關聯對映功能，Oracle 表示該工具將採 Eclipse Public License。此一平台著重設計階段工具並支援所有 J2EE 相容應用軟 體伺服器下的安裝。

Oracle 發展副總裁 Dennis Leung 表示，EJB 3.0 主要改進為其存 續模型（persistence model）的簡化，將一改以往過於笨重的模式， 轉採純 Java 物件（plain old Java objects），使得執行速度加快， 部署更為簡易。

Oracle 日前剛推出其 Oracle Application Server EJB 3.0 Preview， 讓開發者對該規格有更多了解。預期 Oracle 也會推出支援 EJB 3.0 的 Oracle JDeveloper 預覽版本。

Forrester Research 應用軟體開發與整合副總裁兼分析師 John R. Rymer 認為，Oracle 針對 EJB 3.0 的動作反映了該公司策略上的轉 向。他表示過去 Oracle 不曾在這類涉及資料庫的相關標準上扮演主 動角色。Rymer 說該公司認為可以借助 EJB 3.0 標準，並決定在支 援工具的開發上主動出擊。

Oracle 以其 EJB 3.0 的策略，正試圖推展其應用軟體伺服器，該公 司的 Project Fusion 將以該產品為基礎。Project Fusion 將用以 整合 Oracle 的應用軟體，與 PeopleSoft 併購案裡取得的 PeopleSoft 軟體。

Oracle 的 EJB 3.0 專案預料會成為 Eclipse Web Tools 平台的一 部份，Oracle 將會在數月內提供該專案的進一步消息。
