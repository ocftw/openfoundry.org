Broadcom 加入 Linux 行動基金會 推動普及 Linux 手機

建立日期 2007-09-25 16:00      作者是 謝良奇/編譯

通訊晶片大廠 Broadcom 宣佈加入 Linux 行動基金會（Linux Mobile Foundation，LiMo），這是一個由 Motorola、NEC、NTT DoCoMo、Panasonic Mobile Communications、Samsung 與 Vodafone plc 共同成立，為手機開發一致性開放源碼 Linux 軟體平台的產業組織。

Broadcom 表示，Linux 在手機上逐漸獲得重視，該公司將與 LiMo 基金會成員一同解決 Linux 在電力需求、體積與成本上的問題，以便讓 Linux 手持裝置能更為普及。

LiMo 基金會執行董事 Morgan Gilis 表示，Broadcom 加入 LiMo 基金會顯示 Linux 在行動領域逐步受到關注，並彰顯共同產品建構架構上至今所作的努力。

Broadcom 在聲明中指出，3G 裝置愈趨複雜也讓軟體平台必須跟緊腳步。成為 LiMo 基金會夥伴成員後，Broadcom 將能使用該基金會的程式碼，參與工作小組提出貢獻，有助建立在開發智慧型手機的領導地位，同時維持在 Linux 領域的技術領先。

Broadcom 有關手機的半導體產品包括基頻、多媒體應用軟體處理器、電源管理、WiFi、藍芽與 GPS 晶片。加入 LiMo 基金會可以讓該公司預先在這些產品中整合 LiMo 的 Linux 軟體層，並連同該軟體層與該公司的開發套件與硬體參考設計散佈出去。

Broadcom 在日前剛剛擴充了位於台灣的智慧型手機設計中心。該中心以 Windows Mobile 解決方案研發為主。該公司並加入了 Symbian S60 智慧型手機的 S60 開發者社群。

Broadcom 與原出自 Philips 的 NXP 是最先加入 LiMo 的半導體廠。Gillis 預料將有其他廠商跟進。

LiMo 架構委員會主席也是 Samsung 副總裁 Hankil Yoon 表示，Broadcom 多樣化的連接產品將逐漸將更多設備引進 Linux 手持裝置中。Broadcom 行動通訊總經理 Jim Tran 說，Broadcom 在先進通訊產品上，一直跟 Linux 社群有緊密的合作經驗。

與其他手機公司投資了手機作業系統供應商 Symbian 的 Nokia，是目前仍未加入 LiMo 基金會的主要手機製造商。
