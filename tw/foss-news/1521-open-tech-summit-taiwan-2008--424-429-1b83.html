『Open Tech Summit Taiwan 2008 』即將在 4/24-4/29 舉行

建立日期 2008-04-21 23:33      最近更新在 2011-01-06 23:51 作者是 李貞臻

從歐洲到美國，亞洲到台灣等其他國家，有許多公司已經開始使用 Open Source，可是為什麼這麼多公司需要 Open Source 的人？又為什麼這麼多人討論 Open Source？該怎麼知道更多有關 Open Source 的資訊？又該怎麼開始學習 Open Source？那什麼是 Community？

 

不用擔心！今年四月底 OHI 與 ASUS 將共同舉辦一系列的 Open source 交流研討會 -- Open Tech Summit Taiwan 2008，將由有從美國、歐洲和亞洲以及國內的公司解釋他們在 Open Source 的經驗、動機和展望，並以輕鬆且愉快的方式讓參加的人體驗這個史無前例的盛會。希望讓與會者體驗到為什麼世界各地的人會對 Open Source 這麼感興趣。

 

如果很害怕和外國人接觸，沒關係！很怕自己聽不懂，沒關係！主辦單位安排了數次「會前聚會」，讓大家可以率先體驗研討會將會發生的事，並認識什麼是 Open Source 或 Free content。

 

身為協辦單位之一的自由軟體鑄造場 (OSSF) ，在 4/26 下午規劃一個關於自由軟體授權方面的議題，由林誠夏專案經理分享『從版本變遷看自由軟體授權的未來發展』。
