185 期電子報－重要新聞回顧

建立日期 2011-11-21 16:58      作者是 陳瑞霖／編述
自由軟體鑄造場電子報從 2003 年創刊時就一直關注自由軟體界發生的大事。並且依商業性、技術性、發展趨勢等原則，為讀者挑選數則重要自由軟體新聞。經由綜合各方說法及比較不同觀點，讓我們回顧前陣子發生的重要自由軟體新聞吧。

透過釋出開源專案來搶市占率是常見的競爭手法，而今釋出專案的可不是資訊業而是金融業。美國 NYSE Technologies 開源釋出 OpenMAMA 專案，希望能為所有的金融市場資料服務建立標準介面。

關於 GPL 的官司判決又一件案件確定了。德國法院駁回了 DSL 路由器廠商 AVM 針對 Cybits AG 的著作權侵害控訴。新聞後半段另一則也是專利官司，由於不願付授權費，捲入專利訴訟的 Barnes & Noble 控告 Microsoft 從事反競爭行為。

Java 曾經是跨平台開發的選項，如今 Apache Harmony 將停止開發，進入 Attic，象徵一個時代的結束。

開源專案如果出現來自大公司的貢獻會讓特別注意，尤其是向來對開源軟體不友善的公司。Microsoft 針對 Samba 專案貢獻程式碼，引起不少討論。

  1. Linux 基金會和 NYSE Technologies 推出 OpenMAMA 中介傳訊專案
  2. AVM 控訴 Cybits 遭德國法院駁回 Barnes & Noble 要求美國司法部調查 Microsoft
  3. Apache Harmony 終止開發 進入 Apache Attic
  4. Samba 獲得 Microsoft 貢獻的程式碼
