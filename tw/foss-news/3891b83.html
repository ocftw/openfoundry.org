Novell 社群版 SUSE Linux 散佈套件更名 openSUSE

建立日期 2006-08-11 08:00      最近更新在 2007-03-09 00:46 作者是 謝良奇
為了增加辨識度與曝光度，Novell 日前宣佈把 SUSE Linux 的消費版本重新命名為 "openSUSE"。此舉也算是和 Novell 贊助的開放源碼 Linux 專案 openSUSE.org 彼此呼應定調。

Novell 資深副總裁與行銷長 John Dragoon 表示，新的名稱 openSUSE 將社群散佈套件與企業產品區分開來，可以更為顯著地劃分出 不同 Linux 使用族群的兩大市場。 已更名為 "openSUSE" 的 SUSE Linux 目前為 10.1 版，10.2 版預計在今年底前釋出。Novell 這一次更名 openSUSE 的動作，正好把該公司旗下標準企業散佈套件 SUSE Linux Enterprise 與社群為主要用戶的產品線劃分開來，有助用戶更輕易地依需求選擇適當的 Linux 產品。 openSUSE 主要提供 Linux 愛好者與組織做為產品評估，用來獲得對最新 Linux與其它開放源碼技術的初步了解。Novell 將採取在 openSUSE 推出有待測試的新技術，獲得回應加以改善後，再整合到企業級的 Linux 產品中。 openSUSE.org 專案已經有超過 3 萬名會員用戶，SUSE Linux 10.1 的安裝總數最近也打破了 35 萬大關。Dragoon 認為這些亮眼的數字證明成立僅一年的 openSUSE 獲得了相當大的成功、以及 SUSE Linux Enterprise 10 產品贏得的廣大迴響。
