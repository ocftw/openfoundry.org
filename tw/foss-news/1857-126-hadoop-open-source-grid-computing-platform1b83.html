12/6 Hadoop - Open source grid computing platform

建立日期 2008-12-03 18:00      最近更新在 2011-01-06 23:43 作者是 李貞臻

本課程將介紹處理大量資料的問題與挑戰、Grid computing 的核心技術 "MapReduce" 演算法，以及 Hadoop 這個 Apache 軟體基金會旗下的開源 Grid computing platform。 講師為蔡奕楷 ( cuma )，目前任職於 Yahoo!，主要專長：Search、Distributed System、Grid computing。聯絡方式：yikai AT yahoo-inc.ocm。

課程共三小時，詳細的內容時間描述與報名資訊如下：

  • 對象：沒特別限制, 對程式開發或是大量資料處理有興趣者。
  • 人數：每班約 20 人  ( 實機操作 )。
  • 平台：Red Hat Linux ( 依實際狀況再進行調整 ) 。
  • 費用：全程免費，但無供餐。
  • 議程：https://whoswho.openfoundry.org/workshop/details/11-hadoop-open-source-grid-computing-platform-.html
