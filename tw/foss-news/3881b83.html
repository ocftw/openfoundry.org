Nokia 擴大 Eclipse 支援範圍 吸引開發者目光

建立日期 2005-09-20 08:00      最近更新在 2007-03-09 00:46 作者是 謝良奇
行動電話製造商 Nokia 正打算透過加碼對開放源碼的參與度，為其 平台吸引更多開發者，該公司最新的動作是更密切地投入 Eclipse 專案。

Nokia 日前加入 Eclipse 基金會，成為董事會成員以及策略開發者。 Nokia 表示將藉由投入軟體與開發者到新 Eclipse 專案，以支援 Eclipse 開放源碼社群。此外，作為 Eclipse 基金會的策略開發者， Nokia 也將帶領針對行動應用軟體的 Eclipse 平台工具開發，其目 標將是讓該以 Java 為基礎的整合開發環境，能完整支援 J2ME。 Nokia 執行副總裁兼首席技術長 Pertti Korhonen 表示，藉由與 Eclipse 的緊密合作及新的開放源碼行動開發工具專案，該公司將提 供 Forum Nokia 計畫超過兩百萬名開發者，針對 Nokia 平台施行最 佳化的整合工具套件。 Nokia 計畫釋出數種現有 Java 開發工具技術，並積極發展新軟體以催生可供開發基於行動資訊設備概要（Mobile Information Device Profile，MIDP）與連接設備組態（Connected Device Configuration, CDC）的行動 Java 應用軟體。藉由發展用以建構可在其行動電話上 執行之應用軟體，Nokia 希望為其平台吸引更多潛在開發者。 事實上這並非 Nokia 首次涉入 Eclipse 專案，儘管 Nokia 旗下行 動電話一直都有專屬的開發工具，去年該公司卻開始投入 Eclipse，在其 Nokia J2ME 開發者套件（Nokia Developer's Suite for J2ME） 與 Nokia 行動伺服器服務軟體開發套件（Nokia Mobile Server Services SDK）中支援 Eclipse 平台，並為 Eclipse 的嵌入式 Rich Client 平台（Embedded Rich Client Platform）專案提供關鍵開發資源。 Jupiter Research 分析師 Michael Gartenberg 認為，這一切動作 的中心目標都是開發者，該平台越能夠吸引更多開發者、對於使用者的吸引力也就越大。Gartenberg 也表示，Nokia 的策略奏效與否， 現在還很難說，不過接下來可想見 Nokia 會盡其所能炒熱這個市場。
