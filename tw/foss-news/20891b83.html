Canonical 與 Intel 合作 Ubuntu 與 Moblin 將聯手出擊

建立日期 2009-06-02 08:00      最近更新在 2009-06-15 07:05 作者是 謝良奇/編譯
Computex 展覽前夕，Ubuntu 贊助商 Canonical 傳來消息，Canonical 即將宣佈與 Intel、SanDisk 和 RealNetworks 的最新合作。其中，最讓外界感興趣的，或許就是 Canonical 將展出 UNR (Ubuntu Netbook Remix) 的 Moblin 版本。此外，RealNetworks 將授權 RealPlayer for Mobile 至採用 Ubuntu Linux 的 netbooks，以改善 netbook 的使用者體驗。

UNR 是 Ubuntu 8.04 為了儘可能發揮 netbook 有限硬體效能的精簡版本，和多數 Ubuntu 版本不同的是，UNR 的主要對象是 OEM 廠商，而非一般用戶。這個版本的用意是為了讓這一類廠商更容易製造搭載 Ubuntu 且採用 Intel Atom 的 netbook。UNR 的設計特別是為了在 Intel Atom 處理器，配備 512MB 記憶體與 4GB SSD 或硬碟的環境下使用。

UNR 的 Moblin 版本不同之處，就是 UNR 會採用 Moblin 介面，取代 UNR 預設的 Launcher 介面。然而兩者底層皆為 GNOME。

Canonical 的展出內容將以 Intel 現有的 beta 程式碼為基礎。根據消息指出，這個版本不會立即公開。Canonical 也將宣佈，該公司正在開發以 Moblin 2 完整版本為基礎的 UNR。Moblin 2 完整版本根據 Canonical 與 Intel 的說法，即將在年底問世。

目前已有少數 netbook 開始搭載 UNR，例如 SYLVANIA g netbook MESO。UNR 致力於將 Firefox、Thunderbird 與 OpenOffice.org 塞入小小的 netbook 中。UNR 甚至包含了 Adobe Flash、Adobe Acrobat Reader 與 Sun 的 JVM (Java Virtual Machine) 等私有軟體。與其他一般 Linux 散佈套件不同的是，UNR 內含有經過授權，支援 MPEG4 (H.263)、MP3、Apple  AAC 與 Windows Media 的音訊與視訊解碼器 (decoders)。

此外，Canonical 將宣佈與 SanDisk 合作，未來 Ubuntu 在 SSD 上的執行速度可望加快，SSD 上的搜尋速度 (seek speed) 將可媲美 netbook 常見的 5,400 RPM 硬碟機。

最後，RealNetworks 將宣佈授權 Real Player for Mobile 至 Ubuntu 的消息。根據 RealNetworks 經理 Rishi Mathew 的說法，RealPlayer for Mobile 在 netbook 硬體許可下，將支援 HD video。另外，RealPlayer for Mobile 也將支援現有的大多數音訊與視訊格式，儘管其中獨漏 YouTube 採用的 Flash videos (.flv) 與 DVD video。

對此，Mathew 宣稱，對用戶來說，影像解析度與 codec 支援並不是 RealPlayer 在 Linux netbook 上，優於其他影片播放軟體如 mPlayer、VLC 之處。Mathew 表示，缺乏熟悉的媒體播放軟體是 Linux netbook 的高退貨率原因之一。相對地，RealPlayer 從 1995 年開始就廣泛運用在個人電腦上，RealPlayer for Mobile 早先版本則已經搭載在 4.3 億支智慧型手機上。

新的 RealPlayer for Mobile 在功能上，將與 Windows、Linux 桌面和筆電上的 RealPlayer 相近。與 Flash video 相比，Mathew 認為，DVD 支援並非 RealPlayer for Mobile 的優先考量。因為僅有少數 netbook 附有 DVD 磁碟機。 RealNetworks 正設法為 RealPlayer for Mobile 提供 Flash video 支援。

Mathew 指出，採用 Intel 945 Express 繪圖晶片的 Intel Atom N270 netbook，執行 RealPlayer 之時將可在全 WVGA (800x480) 解析度下，支援 hiccup-free 影像。為了達到 1080p (1920x1080) HD 解析度，執行 RealPlayer 的 netbook 必須使用如 Intel 的 Atom Z530 CPU 晶片，並搭配 HD 繪圖加速器。例如，Hewlett-Packard 剛推出的 Mini 1101 即支援了 1080p。

RealPlayer for Mobile 與 codec 套件將立即提供計畫在各種機型上搭載 Ubuntu 的 OEM 廠商。然而，此一版本的 Real Player 將不提供終端使用者購買，僅供納入 netbook 軟體套件之中。
