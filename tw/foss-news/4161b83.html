Red Hat 宣佈 Mono 平台將納入 Fedora Core Linux

建立日期 2006-01-06 08:00      最近更新在 2007-03-08 08:06 作者是 謝良奇
開放源碼開發平台 Mono 即將被納入下一版的 Fedora Core Linux 散佈套件中，基於開放源碼社群對 Mono 的特殊情結，加上 Mono 同時牽動著 Red Hat 兩大競爭對手，Red Hat 表明 Mono 將僅納入非商業的 Fedora Core Linux，並不包括商業版本的 Enterprise Linux。 Fedora Core 是 Red Hat Linux 分支出來的一套非商業 Linux 散佈套件，由 Fedora Project 開發，Red Hat 贊助。Mono 專案由 Novell 主導，企圖發展出 .Net 相容工具，包括 C# 編譯器及共用語言執行環境 (Common Language Runtime)。

Red Hat 桌面團隊經理 Chris Blizzard 日前在他的部落格 (blog) 上表示，Mono 會被納入 Rawhide，也就是 Fedora Core 5 的套件樹中。他表示，他們很高興提供應用其核心桌面平台的另一種方式，Mono 將成為 pygtk、java-gnome for Java 等其它發展技術中的一員。

Mono 專案發起人也是 Novell 主管 Miguel de Icaza 表示，對於 Mono 而言這是很重要的一步，Mono 和 Fedora 用戶將能夠同時使用 Mono，以及採用 Mono 的各種應用軟體。藉由納入 Mono，Fedora 的用戶不需安裝額外軟體，即可使用多種 Mono 應用軟體，如 F-spot 相片管理軟體與 Beagle 桌面搜尋工具。

由於 Mono 旨在發展微軟 .Net 相容工具，不少開放源碼開發者對 Mono 懷抱不信任的態度，Windows 用戶最終能夠執行 Mono/Gnome 軟體這樣的聲音也出現在 Slashdot 上。再加上 Mono 牽動著 Red Hat 與競爭者微軟、Novell 間的關係，Red Hat 已經表明並未計劃將 Mono 引入商業產品 Enterprise Linux中。
