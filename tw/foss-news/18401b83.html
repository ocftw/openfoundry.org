Canonical、ARM 合作 Ubuntu Desktop Linux 移植 ARMv7 架構

建立日期 2008-11-18 08:00      作者是 謝良奇/編譯

Ubuntu Linux 商業贊助廠商 Canonical 宣佈 Ubuntu Desktop Linux 將移植到 ARMv7 架構上。鎖定 Netbook 的 Ubuntu ARM 散佈套件，可能將會讓 Intel 失去過去 30 年來 x86 一路甩開其他架構所依賴的軟體優勢。

正當 Intel 致力於以基於 x86 的低功率 Atom Z500 處理器晶片家族，在行動設備市場迎擊 以 ARM 為基礎的處理器之時，ARM 打算轉而在 Netbook 市場中，積極開發一完整平台以便動搖 Intel 的地位。

Canonical 宣佈將開發針對 ARMv7 架構進行最佳化的 Ubuntu Desktop Linux 散佈套件完整版本。此一  ARMv7  移植版本將鎖定基於 ARM Cortex-A 處理器核心的系統單晶片 (system-on-chips，SoCs)。此外，該散佈套件很可能也會支援未來由 ARMv7 架構授權者建構的核心內建 SoCs。

桌面與 Netbook 使用的 Ubuntu ARM 散佈套件將於 2009 年 4 月公開推出，Canonical 特別提到以 ARM Cortex-A8 以及更新的多核心 Cortex-A 為基礎的 SoCs 與 Netbook，將是此一散佈套件的主要對象。

ARM 表示此一發展起於設備製造商的需求。此外一套完整的軟硬體解決方案也將增加台灣 ODM 廠商開發基於 ARM 平台的 Netbook 設備的機會。

對此新的 ARM 散佈套件 Canonical 並未透露太多詳細資訊，僅僅表示該公司與 ARM 的合作將會凸顯以及利用兩家公司對於各開放源碼計畫的支持，這些計畫包括 Linux 核心、Debian、GNOME 與 Mozilla 社群。ARM 過去主動支持了上述專案在 ARM Linux 上的開發工作。

日前 ARM 行動解決方案主任 Rob Coombs 曾表示，ARM 的高階 Cortex-A 處理器家族核心不久後將在 Netbook 上現身。未來基於 ARM 核心的 Netbook 將執行 Linux、Windows Mobile、Windows CE，但無法執行 Windows XP 或 Vista，除非微軟肯移植 Windows XP 與 Vista 到此一架構上。

不少桌面 Linux 散佈套件商都已經推出 Netbook 版本，包括 Mandriva 的 Mandriva Mini，Canonical 可說在擴展 Ubuntu 到 Netbook 與行動裝置上採取特別謹慎的態度。

2007 初，Canonical 推出 UME (Ubuntu Mobile and Embedded) 專案，今年 6 月 Canonical 則與 Intel 贊助的 Moblin 專案合作，推出基於 UME 的 Ubuntu 8.04，名為 Ubuntu MID Edition。

此外，Canonical 也對其散佈套件推出一 Ubuntu Netbook Remix  版本。9 月 Foresight 集團採用 Ubuntu Netbook Remix 作為其最新 Foresight Linux 散佈套件，名為 Foresight Mobile Edition 1.0。
