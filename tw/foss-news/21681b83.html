Google 釋出 Chrome 3 Mac、Linux 開發版本更新

建立日期 2009-09-24 08:00      最近更新在 2009-10-14 01:26 作者是 謝良奇/編譯

Google 釋出了旗下的 Chrome 瀏覽器最新版本，並且承諾 Linux 與 Mac 版本將會在年底前與使用者見面。隨後，Google Chrome 團隊也釋出開發者版本的更新版本，在各平台上的效能皆有提升，其中，Mac 與 Linux 用戶可以體驗到最大的不同與新功能。

Chrome 第 3 版提升了速度，Google 宣稱加強後的 Javascript 效能比起上次的穩定釋出 Chrome 2，有超過 25% 以上的提升，跟第 1 個版本相比，則有 150% 以上的改進。

新增加的新分頁 (New Tab) 頁面功能，讓用戶可以重新排列最愛網站的縮圖。結合搜尋列與網站位址列的 Omnibox 這次也有所改變。Google 修改了 Omnibox 的下拉式選單，增加了小圖示以茲區別推薦網站、搜尋、書簽與瀏覽記錄的網站。

這次 Chrome 增加了更多 HTML5 的功能。Google 在部落格中表示，他們對 HTML5 中的 <video> 標籤，特別感到興奮。這個標籤讓影片嵌入跟嵌入一般圖片一樣簡單。<video> 標籤不須 plug-in 即可播放影片。在外觀上，新版本加入了 Themes，允許用戶新增顏色、樣板與圖片。

新釋出的開發者版本更新中，各作業系統上的版本都經過部份調整，包括改進 New Tab 載入時間，以及修改 Chrome 處理延伸套件的方式。

Mac 用戶，特別是採用 Snow Leopard 的用戶，將會樂於見到，匯入 Firefox 設定的功能變得更為順暢，搜尋引擎管理器也出現在 Mac 版本中，此外也有與 Snow Leopard 相關的臭蟲修改。Linux 用戶可以體驗到改進後的書簽系統，AIM: links 等呼叫外部服務的 URLs 的處理方式也有所改善，對 Flash 影片也有更好的處理。

讓人等待已久 Chrome 的 Linux 與 Mac 版本，已經接近完成，並且應該會在年底前推出。產品經理 Brian Rakowski 表示，他們將這兩個版本放在開發者頻道中已有一段時間，希望確保在這些平台上提供令人滿意的原生體驗，並符合該公司在穩定性與效能上的標準。
