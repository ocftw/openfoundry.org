Oracle 提供 Ajax 開放源碼工具 簡化 Web 2.0 開發繁瑣步驟

建立日期 2006-05-23 08:00      最近更新在 2007-03-08 08:00 作者是 謝良奇
Oracle 將一系列用以開發 Ajax 使用者界面的工具軟體，提供給開 放源碼社群。這些工具可以讓開發者得以自動產生 Ajax 界面，免去 自行開發的困擾。 Ajax 讓開發人員可以發展預先提取資料的線上應用軟體，藉以提高 軟體的互動性與可用性，也可大幅減少使用者操作軟體時必須重複載 入的網頁數量。

Oracle 伺服器技術發展資深副總裁 Thomas Kurian，日前在舊金山 召開的 JavaOne 大會上發言時表示，利用 Oracle 貢獻社群的這些 工具，開發者可藉此建構界面完善的應用軟體。

Oracel 更認為開放源碼工具和標準是簡化下一代互動式網路應用軟 體，也就是俗稱 Web 2.0 網站的關鍵資源。

Oracle 首席架構師與工具與中介軟體副總裁 Ted Farrell 就表示， Java、開放源碼與腳本語言 (scripting languages) 會是新一波 Web 應用軟體開發的關鍵元件。
