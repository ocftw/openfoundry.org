Dell 加入微軟、Novell 合作聯盟

建立日期 2007-05-16 15:17      最近更新在 2007-07-03 23:27 作者是 謝良奇
Dell 宣佈將成為微軟與 Novell 成立的 Windows-Linux 合作聯盟中，第一個主要系統供應商。按照協定，Dell 將會從微軟手中購買 Suse Linux 企業伺服器的認證。

去年 11 月微軟與 Novell 同意共同開發與行銷讓 Windows 與 Linux 更緊密整合的產品。微軟根據其合作內容，同意保證 Novell 的 Suse Linux 企業用戶可免於專利權利主張之外。微軟並且負責行銷與販售 SUSE Linux 企業伺服器維護與支援認證的角色。

Dell 是第一個加入此聯盟的硬體廠商，Dell 表示希望能藉此滿足用戶對於 Windows 及 Linux 整合上的需求。Dell 宣稱用戶對於微軟及 Novell 合作表達的強烈興趣，是這項決定背後的關鍵因素。

在合作條款下，Dell 會針對 Suse Linux 企業版成立用戶行銷團隊，鎖定 Dell Linux 用戶之外的 Linux 使用者，鼓勵 Dell Linux 用戶之外的 Linux 使用者改用 Suse Linux 企業伺服器。至於 Dell Linux 用戶，因為 Dell 已經提供紅帽 (Red Hat) Linux 給企業用戶，現有的用戶將不會是這波新的行銷動作的鎖定目標。

有關合作的財務細節則並未透露。這並非 Dell 首次與這兩家公司合作，Dell 目前與微軟、Novell 都有個別的合作關係。

從微軟與 Novell 發表合作後，兩家公司已經成功地達成 4 萬個 3 年期 Suse Linux 企業伺服器支援認證。其中最大的客戶之一，就是沃爾瑪百貨商店 (Wal-Mart Stores)。
