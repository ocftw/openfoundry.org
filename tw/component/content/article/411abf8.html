OSDL 調查：「使用者主動要求」為桌面 Linux 部署首因

建立日期 2005-12-09 08:00      最近更新在 2007-03-08 08:03 作者是 謝良奇
開放源碼發展實驗室（Open Source Development Labs，OSDL）日前 舉辦的桌面 Linux 調查（Desktop Linux survey），回覆多達 3300 封。該調查目的在於了解 Linux 進駐桌面的關鍵議題，以及提高 Linux 桌面系統接受度的主要障礙。 OSDL 主要分析員 Dave Rosenberg 表示，最讓他們訝異的是使用桌 面 Linux 的第一、二項主因居然不是總持有成本（TCO）或安全性， 而是員工要求以及競爭者已先成功部署 Linux。Rosenberg 說，外界 一向認為 Linux 基本上是開發者工具，不會從競爭的角度加以看待。

使用桌面 Linux 的第三、四項主因為 TCO 與安全性。Rosenberg 認 為 TCO 與授權費用縮減這類經濟性優勢，其順序之所以比安全性高， 可能是因為安全性已經到達一個穩定狀態，TCO 與授權費用則相對有 正面進展。OSDL Linux 桌面計畫經理 John Cherry 同意這種看法， 並認為該項調查如果在一兩年前舉行，會得到相當不一樣的結論，這 一點顯示 Linux 給人的觀感已經在改變，企業使用 Linux 不只是因 為成本，而是因為 Linux 真的好用。

至於使用者認為成功 Linux 部署最關鍵的應用軟體，依次為電子郵 件、辦公室套件以及網路瀏覽器。Rosenberg 表示電子郵件所以高居 榜首，可能是因為無論何種平台電子郵件都是一種殺手級應用軟體， 或者是，Linux 還是缺少高品質的電子郵件軟體。

受測者認為讓用戶不願意改用 Linux 的主因，是應用軟體及周邊支 援。調查報告指出受測者希望 Linux 能支援現有與新的非開放源碼 桌面應用軟體與工具如 VPN client。

被問到考慮採用或正在使用的 Linux 散佈套件，Ubuntu 以 53% 勝 過 Open SUSE 的 32%、Novell/SUSE Enterprise Linux 的 28% 及 Debian 的 27%。Ubuntu 勝出的因素包括：釋出頻繁、龐大社群支援、 對於硬體的相容性高。
