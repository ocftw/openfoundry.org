Intel 推出 LessWatts 計畫 節省 Linux 系統電力消耗

建立日期 2007-09-27 16:00      作者是 謝良奇/編譯

目前已經有不少用來節省電腦電力消耗的技術，Intel 將可以降低 Linux 伺服器、個人電腦與設備電力消耗的開放源碼軟體與相關建議結合，推出 LessWatts.org 計畫。

在 Intel 開發者論壇（Intel Developer Forum，IDF）上由該公司軟體與解決方案群副總裁兼總經理 Renee James 詳細介紹了 LessWatts，這是一個針對程式開發人員與系統管理者等技術專家的計畫，其中包括了多個 Intel 專案。例如 PowerTop。PowerTop 能找出佔用處理器，使處理器無法進入低功率睡眠狀態的軟體。

Intel 首席 Linux 與開放源碼專家 Dirk Hohndel 表示，囊括了相當多現有技術是 LessWatts 計畫有趣之處。LessWatts 是一個開放源碼專案，他們希望建構針對分享節省電力技巧、相互協助並提高此議題聚焦程度的社群。

Hohndel 表示，使用 LessWatts 提供的建議與軟體工具，可以為目前的雙處理器伺服器省下約 10 瓦的電力。雖然乍看並不多，但每節省一瓦，就可以連帶省下用來冷卻處理器耗費的 1.3 瓦電力，加上考慮到大型資料中心動輒數以千計的伺服器，如果全數採用這些節能方法，能節省下來的電力將十分可觀。

雖然 LessWatts 包含的技巧與軟體只針對 Intel 晶片進行測試與開發。Hohndel 認為這個專案很可能影響到包括 AMD 在內的其他廠商。Hohndel 表示，LessWatts 是一個社群開放源碼專案，自然不會只限制在 Intel 上。Linux 廠商紅帽（Red Hat）與 Novell 都已加入這個計畫。

除 PowerTop 軟體外，LessWatts 提出不少用來降低 Linux 電力消耗的技巧和技術，包括變更 Linux 排程器的核心元件、壓縮圖形晶片與處理器通訊資料、修改高階軟體、降低網路傳輸率以及免計時 Linux 核心（tickless Linux kernel）。

部份技術已經實作到目前的 Linux 核心中，有部份則需要使用額外工具與核心修補程式。下一個版本的 Fedora、OpenSuse 與 Ubuntu，預料將會加入部份技術。

LessWatts 僅針對 Linux 用戶，Hohndel 解釋說該計畫採用的技術雖然並非不可能，但要移植到其他作業系統並不容易。例如免計時 Linux 核心是針對 Linux 核心的修改，並不容易進行移植。
