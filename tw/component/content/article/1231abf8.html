BT 加入 Linux 電話標準論壇 Linux 行動裝置發展廣受關注

建立日期 2007-10-02 16:00      作者是 謝良奇/編譯
Linux 電話標準論壇 (Linux Phone Standards Forum，LiPS) 宣佈全球電信與營運商 BT 加入該協會，將參與此一組織的標準化與規格制定作業。LiPS 因為 BT 的加入，連同 France Telecom/Orange、Telecom Italia 等會員，將使全球營運的版圖更為完整，進一步達成該論壇為 Linux 手持裝置帶來高價值應用軟體與服務的目標。

LiPS 於 2005年 11 月成立，為了加速 Linux 在固定 (fixed)、行動與整合型 (converged) 裝置上的普及率，進而標準化可直接影響應用軟體與使用者服務部署、開發與相互整合能力的 Linux 服務及 API。創立成員包括 ARM、Cellon、Esmertec、France Telecom/Orange、FSM Labs、Huawei、Jaluna、MIZI Research、MontaVista Software、Open-Plug 與 PalmSource。

LiPS 已經有包括營運商、晶片與設備製造商及軟體開發者等多達 24 家成員，卻不見歐洲與美國通信大廠的蹤跡，若干通信領導廠商更另行成立了 LiMo 基金會。

鑑於 BT 這樣的全球電信與營運商，能直接與最終使用者接觸，並且貼近市場需求進展，加上 BT 支援 170 個國家國際與跨國組織所代表的全球可見度，BT 對於市場的服務與設備能力需求有著獨一無二的洞見。LiPS 論壇總裁 Haila Wang 表示，LiPS 歡迎 BT 也期待該公司對 LiPS 工作小組所能帶來的貢獻。BT 跨越數十個市場的全球電信經驗以及面對數以億計使用者的服務經驗，也有助於 LiPS 的標準化工作。

BT 技術長 Matt Bross 說，BT 致力於跟隨客戶的步伐進行創新，為此，他們必須不受組織的侷限，融合內部與外部的創新，而 LiPS 論壇正是這種開放創新的關鍵。

透過標準化，LiPS 論壇將有助 BT 降低部署與管理成本，以更為快速、高成本效率的服務部署及更具吸引力的用戶方案維持競爭優勢，並支援 Linux 裝置上可相互整合的創新服務。

近日由於 Linux 軟體公司 MontaVista 加入由 Motorola 等主要手機製造商成立，以開發及推廣 Linux 手機平台為主的 LiMo 基金會，加上開放手機聯盟 (Open Mobile Alliance) 與 LiPS 宣佈合作，使得 Linux 在行動裝置的應用格外受到關注。
