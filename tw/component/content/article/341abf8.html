MontaVista 執行長專訪 暢談嵌入式 Linux 發展

建立日期 2004-05-27 08:00      最近更新在 2007-03-09 01:17 作者是 謝良奇
LinuxDevices.com 日前專訪 MontaVista 執行長 Jim Ready，談及他針對他口中「世紀之戰」，如何為其公司鋪路。專訪中包含現今的計算趨勢，如 Linux 逐漸形成跨越企業伺服器到嵌入式設備的端對端（end-to-end）作業系統，嵌入式 Linux 成為手機的標準平台，嵌入式 Linux 以及整個嵌入市場的競爭現況，Linux 標準組織的工作，MontaVista 的市場策略等等。

Ready 認為，如今微軟與 Linux 之間，好比一場世紀之戰。微軟，以及另一方 Linux，都有龐大的金錢資源。MontaVista 正位於中心。他並不看好 Wind River 和 Red Hat 之間的結盟，並且認為資源分配以及非純粹 Linux 式的組合，是其弱點所在。對於 Linux 從企業到嵌入式領域作為終端對終端解決方案的成長表現，Ready 表示 MontaVista 在此一市場中，已經佔據有利地位。Eric S. Raymond 曾經用市集與教堂，來比喻開放源碼和私有軟體。對於以此來比喻 Linux 和微軟，Ready 認為，開放源碼發展實驗室（Open Source Development Labs，OSDL）及其 Carrier Grade Linux，已為開放源碼發展帶進半教堂式的氣氛，成為組織化的市集。MontaVista 也是 OSDL 及消費性電子 Linux 論壇（Consumer Electronics Linux Forum，CELF）的成員，Ready 相當樂見於 OSDL、CELF 這些標準組織的出現。Ready 對嵌入式市場的演進，自有一番看法。他認為，目前嵌入式市場就像 PC 出現前的計算環境，只有專用型電腦等垂直方案，直到標準化平台（IBM PC 和 DOS）出現才改變這一切。他認為這個市場將趨於水平化，智慧型手機會成為客製化平台。他說，假如你拿的是微軟的手機，你很可能會看到死當的藍色畫面。嵌入式 Linux 作為一項技術，Ready 表示仍有需要加強的不足處。而作為一個市場看待，他認為其成長率將當驚人，該市場有相當強勁的需求。他表示智慧型手機會是嵌入式 Linux 的關鍵契機，圍繞在其上的平台將成為兵家必爭之地。家用領域則是另一處重要的關鍵，像是數位電視、機上盒等。他將即時功能，視為嵌入式 Linux 技術面臨的挑戰。目前有多方努力正在進行，包括 MontaVista 和 OSDL，OSDL、CELF 會是整合各方技術力量的優秀途徑。Ready 表示自己難以預估嵌入式 Linux 市場的成長。他認為 Linux 未來不會走向高度分裂，團結是 Linux 的承諾。嵌入式 Linux 現象已經無法停止，未來會是異常光明的，而這就是他所能說的一切。
