Intel 與 台灣合作共為開放源碼 Moblin 成立發展中心

建立日期 2008-11-06 08:00      作者是 謝良奇/編譯
Intel 的開放源碼 Linux 行動平台計畫有了多家 Linux 散佈套件商的支持挹注，近來聲勢不小。此一計畫名為 Moblin，旨在針對行動網際網路裝置、小型筆記型電腦 (subnotebooks)，以及設計運用 Intel  Atom 處理器的整合式車輛運算系統，建構最佳化的 Linux 堆疊。日前 Intel 與台灣政府單位合作，計畫建立協助硬體廠商於 netbook 與行動網際網路裝置上，運用此一 Moblin 作業系統的發展中心。此外，Intel 更投資台灣行動電信業者 VMAX，計畫部展 WiMAX 網路。

全球最大晶片製造商 Intel 針對使用 Intel Atom 處理器的裝置發展了 Moblin。該開放源碼軟體內含 Linux 核心、使用者介面、瀏覽器、開發工具與其他資源。Moblin 的設計提供快速的開機速度、利於手指操作 (finger-friendly) 的使用者介面、低電耗、記憶體用量小等特點。最初基於 Ubuntu 設計的 Moblin，已轉為 Fedora 系統並採用基於 RPM 的套件模式。

Moblin 的設計允許主要 Linux 散佈套件商採用其元件並整合到自有的軟體平台中，數家商業 Linux 公司已經承諾發展以 Moblin 為基礎的散佈套件，如 Novell、Mandriva 與 gOS 的散佈套件商 Good OS。

新成立的發展中心座落於台灣，將使得當地裝置製造商，如 Acer 與 Asustek Computer，可在其產品中應用該軟體。目前來自這兩家公司的 netbook，採用的是 Windows XP 與 Linux 作業系統。新的 Moblin 中心有意讓 Moblin 更廣泛地應用於 netbook 與其他可用以連接網際網路的小型裝置中。

Intel 執行長 Paul Otellini 表示，電腦產業當今最熱門的就是 netbook 現象，而這股風潮的起點正是台灣。台灣經濟部 (Ministry of Economic Affairs) 成為 Moblin 最新的支持組織，經濟部長尹啟銘表示，新的 Moblin 發展中心將於 12 月成立。

Linux 社群知名人士將行動網際網路裝置與 netbook 市場，視為 Linux 的一大機會。Linux 基金會 (Linux Foundation) 總裁 Jim Zemlin 日前即表示，低授權成本與高度彈性，將使開放源碼 Linux 作業系統成為主導此一領域的強力競爭者。他更指出 Moblin 為最開放且最具社群親和性的行動 Linux 計畫之一。

儘管如此，Linux 在 netbook 與行動網際網路裝置的成長，仍須跨越數道障礙。首先硬體廠商仍須解決整合議題，面對驅動程式與達成適當可用性的困難。Intel 的 Moblin 計畫便為此若干挑戰，集合 Linux 散佈套件商與硬體廠商提供共用堆疊，針對共用硬體元件提供驅動程式，並為具備小螢幕與觸控輸入特性的行動裝置，提供適用的上層使用者介面元件。

Intel 與台灣經濟部共同成立的 Moblin 實驗室，將為有意開發採用 Moblin 平台產品的軟硬體廠商，提供訓練與諮詢服務。Intel 相信如此將加速 Moblin 普及率，並協助台灣硬體公司加速產品上市速度。

另外，Paul Otellini 提到該公司的投資部門 Intel Capital，將在台灣電信業者 VMax 上，投資 1150 萬美元，協助部署其第一個 WiMax 網路。該網路預期明年上半年正式推出。Intel 本身即是第四代無線寬頻技術 WiMax 的主要支持者。

Intel Capital 對於 VMax 的投資，反映出 Intel 對於加速全球 WiMax 部署的重視。根據 Intel 的資料，透過 Intel Capital，Intel 至今已投資 30 家 WiMax 技術公司與服務供應商。Otellini 在聲明出指出，WiMax 是 Intel 啟動真正行動網際網路策略的一項關鍵元件。Intel Capital 對 VMax 的投資將締造台灣第一個 4G 無線網路。

Intel 為了推展行動 Linux 普及並且讓 Moblin 成為裝置製造商的堅實平台，已經投入許多資源。如果 Moblin 真能解決許多困擾 Linux 裝置製造商的問題，並提供一個凝聚性更高的平台給以 Atom 為基礎的裝置，對於此一開放源碼作業系統，將是一大勝利。
