新釋出 Opera 10 提升瀏覽速度

建立日期 2009-09-03 08:00      最近更新在 2009-09-13 18:23 作者是 謝良奇/編譯

Opera Software 日前釋出了 Opera 10，作為該公司桌面瀏覽器重要升級，內建的網頁壓縮與重新翻修過能自動顯示頁面略圖 (thumbnails) 的分頁列 (tab bar)，是其主要特色。這款免費的瀏覽器，可由該公司網站下載針對 Windows、Mac OS X 與 Linux 平台超過 40 種語言的版本。甚至在 Apple 新推出的 Snow Leopard 作業系統上也可運作。

該款瀏覽器的即時壓縮引擎 Opera Turbo，能加速低速網路連線的瀏覽，該公司宣稱，與競爭對手相較，有 8 倍的效能改進。Opera 執行長 Jon von Tetzchner 表示，Opera Turbo 是他們的最新創新，每個人都應該試試，因為每個人都可能遇到網路連線過慢的情況。

Opera 瀏覽器這次在使用者介面上也有顯著的改變。翻新的分頁列可以透過點選與拖拉改變大小，更重要的是，Opera 會顯示出各分頁的內容略圖，使用者可以將分頁列從預設的上方位置，移到瀏覽器視窗側邊，以因應寬螢幕顯示。

後來被 Google Chrome 瀏覽器採納的 Speed Dial 功能，此次也做出改進，擴展後的 Speed Dial 能顯示比過去 Opera 瀏覽器更多的時常造訪網站，甚至能夠以背景圖片加以客製化。

本次釋出的缺憾之一，是 6 月中發表的技術平台 Opera Unite，藉由在 Opera 中加入袖珍的網路伺服器，讓使用者無須使用第三方的點對點服務，即可相互分享檔案、照片與音樂。儘管在發表當時該公司似乎有意將 Unite 加入 Opera 10，但在這次釋出的 Opera 10 中卻不見其蹤影，甚至有傳言 Unite 的安全性不佳。外界一開始對於 Unite 的興奮熱度，已經有逐漸冷卻的跡象。Opera 並未針對 Opera Unite 的時間表做出立即回應。
此外，Opera 公司年初發表的 Carakan JavaScript 引擎，也未納入 Opera 10 之中。Carakan JavaScript 引擎宣稱比 Opera 10 alpha 所採用的 Futhark 引擎快上 2.5 倍之多。雖然相較於 Firefox，Opera 10 有較好的效能表現，但是 Google Chrome 特別是在大量採用 JavaScript 的網站上，如 Gmail 與其他 Web 2.0 網站，要比 Opera 來的快。

Opera 10 即使新增諸多功能，但要說服其他瀏覽器使用者改用 Opera，並非如此簡單。根據美國網路數據調查公司 Net Applications 的資料，Opera 目前佔有 2.1％ 的全球瀏覽器市場，落後於微軟 Internet Explorer (IE)、Mozilla Firefox、Apple Safari 與 Google Chrome 等瀏覽器之後，為第 5 名的瀏覽器。

根據 W3Schools 的統計資料顯示，Opera 的市佔率從今年初的 2.3%，下滑至目前的 2.1%。Opera 10 雖然有許多不錯的功能，介面也有所改善，但這些改進並不足以打造出比 Firefox 或 Chrome 更好的瀏覽器。反觀 Firefox 以開發者體系與多樣化功能著稱，Chrome 則著重在速度與核心功能上。

事實上，Opera 今年因為歐盟針對微軟的反壟斷 (antitrust) 案件，而登上新聞版面的次數，要多於該公司的技術發表。Opera 在 2007 年向歐盟監管機構針對 IE 壟斷提出的投訴，造成歐盟在 2009 年 1 月控訴微軟藉由在 Windows 中隨附 IE，非法保護其免於競爭。今年夏天，微軟表示願意在 Windows 中加入投票畫面 (ballot screen)，讓使用者選擇包括 Opera 在內的至少 5 種瀏覽器。歐盟至今仍未就微軟的提案做出最終決定。
