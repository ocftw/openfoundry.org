Yahoo 投入搜尋開發者社群建構

建立日期 2005-03-18 08:00      最近更新在 2007-03-08 08:31 作者是 謝良奇
面對競爭對手 Google 已行之有年，將產品 API 加以公開的策略， 知名網路公司 Yahoo 日前終於推出了 Yahoo 搜尋開發者網路（Yahoo Search Developer Network），以更多的 API 搜尋與功能，企圖拉 攏開發者社群。 Yahoo 搜尋開發者網路將 Yahoo 搜尋產品及 Overture 的 API 提供 給開發者使用。此套 API 可讓網站開發者借助 Yahoo 的搜尋服務， 整合至自身系統，其功能包括了圖片、影片、新聞及本地端搜尋。未 來也會提供相關搜尋與拼字訂正的 API。該網路也提供了範例程式及 相關工具，囊括了 Java、JavaScript、Perl、Python 與 PHP 等語言。

這組 API 有每日使用次數上的限制。Yahoo 表示每個開發者每天可 呼叫 5000 次 API，這個數目是 Google 的 5 倍之多。

Yahoo 此一動作的主要目的，是希望開發者藉著具備郵件論壇、討論 群組等功能的互動線上社群，互相分享知識。該線上社群尚具備了應 用軟體展示、部落格（weblog）以及 wiki 系統，開發者可透過上述 系統發問、交流，並針對這些 API 提供回應。Yahoo 發言人 Stephanie Ichinose 表示，他們相信此計劃將於開發社群內，培育 新的生態系統。

這類非商業性計劃會為 Yahoo 或 Google 帶來多少好處仍不得而知， 以 Google 而言，這樣的作法可為該公司品牌加分，同時有助該公司 發掘有潛力的開發者。對於 Yahoo 來說，競爭對手 Google 早已著 手建立與開發者間的良好關係，難怪 Yahoo 此時也推出相類似的計劃。

Yahoo 搜尋開發者網路也提供開發者 Overture 的 API。Overture 這個 API 於 2001 年起就開始提供給其合作夥伴使用，隨著這項計 劃的宣佈，Overture 也更名為Yahoo 搜尋行銷解決方案（Yahoo Search Marketing Solutions）。此一計劃可讓搜尋網站商、廣告客 戶與第三方開發者建立與 Overture 產品可直接互動的系統，以管理 和分析搜尋行銷活動。
