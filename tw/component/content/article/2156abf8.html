歐盟開啟 Oracle-Sun 購併調查

建立日期 2009-09-08 08:00      作者是 謝良奇/編譯

歐盟主管機關日前暫緩 Oracle 對於 Sun Microsystems 所提出的購併，指出這項合併有可能阻礙 Sun 所擁有的知名自由資料庫軟體 MySQL 的發展。歐盟並啟動反壟斷調查，表示希望確保 Oracle 持續開發 MySQL。

這個價值 74 億美元的購併案，在美國司法部 (Justice Department) 准許通過後，由歐盟執行委員會 (European Commission) 所下的延長調查決定，看來格外敏感。美國主管機關雖然針對 Oracle 在其部份業務領域的市場力量有所疑慮，在開放源碼軟體上的顧慮，卻不如歐盟。

過去歐盟執行委員會也曾與美國司法部意見相左，例如歐盟執行委員會曾在美國當局核可後，駁回 General Electric 與 Honeywell 的合併。當時的歐盟競爭委員會專員 (competition commissioner) Mario Monti 表示，G.E. 在飛機引擎市場可能過於獨霸。近年來，Monti 和目前的競爭委員會專員 Neelie Kroes，就是否強制微軟更改其 Windows 作業系統上，也與美國當局的意見有所分歧。

Oracle 曾就其多項購併與歐盟機關進行斡旋，但最後總能如願過關。2004 年，在歷經徹底調查後，該委員會無條件通過 Oracle 對 PeopleSoft 的購併。美國司法部雖然反對這項購併，但卻輸了與 Oracle 的訴訟。之後歐盟委員會也同意無條件通過了 Oracle 對 Siebel Systems 的購併。

Kroes 在宣佈這項決定時警告，這個購併可能傷害 Sun 持有的重要軟體產品的發展。而此一產品 MySQL，卻是全球廣泛應用的企業資料庫，同時也是 Oracle 軟體產品的競爭對手。她表示，包括歐洲在內的全球多數地區，正當從經濟谷底復甦之際，確保開放源碼軟體的存續實為重要。

Kroes 認為，在當前的經濟情勢下，所有企業都在尋求高成本效益的 IT 解決方案，以開放源碼軟體為基礎的系統正逐漸崛起為私有解決方案的替代性選擇。她說，延長調查是希望確保這些替代方案能夠持續存在。

歐盟執行委員會在 2010 年 1 月 19 日前，有 90 天的時間，決定要通過此購併案或予以阻擋。委員會通常會要求企業透過售出部份業務等動作，減低壟斷疑慮。

歐盟官方表示，由於 MySQL 在持續加入功能並吸引更多用戶，將對 Oracle 領導市場的資料庫軟體構成威脅，他們將調查 Oracle 進一步發展 MySQL 開放源碼資料庫的意願。

儘管有諸多考量，部份專家認為 Kroes 的疑慮可能並無事實根據。IDC 一位分析師 Bo Lykkegaard 表示，Oracle 購併 Sun 有許多理由，MySQL 並非重點。Lykkegaard 認為，甚至 Oracle 藉由維持 MySQL 的開放源碼特性，能夠發展部份業務，接觸到小型公司或是大型企業中需要低成本軟體以執行非關鍵任務的部門。
