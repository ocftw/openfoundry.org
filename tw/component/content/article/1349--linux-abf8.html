分析中國嵌入式 Linux 市場需求

建立日期 2007-12-27 08:00      最近更新在 2011-01-12 19:32 作者是 Linuxpilot

較早前舉行的國際資訊科技博覽中，大陸許多省均派出代表團出席交流，Linuxpilot 把握機會訪問了江蘇 Linux 公共技術支持中心主任趙剛，從而了解中國嵌入式 Linux 市場的最新發展，希望本地的廠商可及早洞悉先機作出部署。

◎ 中國政府支持發展自主軟體產業

從第一個 Linux 核心誕生到現在，世界 Linux 技術發展突飛猛進，基於 Linux 的應用日益廣泛，其開放、安全、穩定的特性得到愈來愈多用戶的認可。目前中國 Linux 產業高速發展，中國政府對於 Linux 的產業支持政策進一步明朗，以 Linux 為契機，發展中國的軟體產業，已經成為各級政府領導和產業界人士的共識。而目前，隨著 Linux 在中國的高速發展，Linux 的產業化進程也日趨深入。但在快速發展的同時，制約因素也逐步顯現。這些制約突出表現為：缺乏與國際 Linux 權威組織的有效介面；Linux 的人才培養匱乏、商業發展模式很窄以及缺乏相應的標準；Linux 的高新技術研究缺乏合作等。

◎ 江蘇省 Linux 市場需求龐大

江蘇省在科技領域，擁有眾多的高等院校和科研機構，為江蘇 Linux 產業發展提供堅實的人才和技術基礎；在經濟領域，江蘇以電子、資訊技術為主導產業的格局，為 Linux 產業發展提供了巨大的市場需求。然而江蘇的 Linux 產業發展目前落後於北京、廣東、上海。北京不僅投入鉅資扶持 Linux 相關企業，在 Linux 應用方面做出了積極的推動作用；廣東省投入六千多萬元的鉅資，成立了廣東省 Linux 公共服務技術支持中心，統一規劃廣東省的 Linux 產業發展；上海通過出資引入 Linux 技術研發公司，在短時間內將上海 Linux 產業規模提升到了相當的高度。在此情況下，建設江蘇 Linux 公共技術支持中心就顯得尤為必要。

◎ 打造江蘇 Linux 產業品牌

基於以上共識，江蘇軟件園和新華科技（南京）系統軟件有限公司決定利用各自優勢資源，共同建設江蘇 Linux 公共技術支持中心，中心籌劃工作於 2004 年 9月開始，中心正式成立於 2005 年 12 月。江蘇軟件園作為中國重點建設的 11 大軟體產業基地之一，具有顯著的政策、金融、技術等平台優勢，已經成為江蘇沿江軟件產業帶的龍頭園區，為推動江蘇省軟件產業的發展發揮了積極應用。新華科技系統軟件有限公司（SWHSS）作為中國 Linux 產業最具代表的企業，在 Linux 技術、教育培訓、國際合作、軟體出口方面具有豐富的資源和成功的運作經驗。建成後的江蘇省 Linux 公共技術支持中心將由 Linux 國際合作中心、Linux 教育培訓中心、Linux 軟體外包出口中心、Linux 就業中心、Linux技術應用中心、Linux 商業孵化中心和產業資本投資中心及中國 Linux 標準研究中心組成。該中心將充分整合 Linux 產業領域的政府資源、企業資源和國際合作資源，在中國政府的大力支持下，突出江蘇軟件園的產業引導作用，發揮新華科技在技術、市場和資本運作領域的優勢，致力於 Linux 國際合作、教育培訓、軟體外包、技術研發等方面的發展，打造江蘇 Linux 產業的優勢品牌。

◎ 江蘇 Linux 支持中心成立背景

江蘇省 Linux 公共技術支持中心是在江蘇省政府、江蘇省發展改革委員會、江蘇省科技廳、江蘇省資訊產業廳、江蘇省外貿廳大力支持下成立。江蘇省 Linux 公共技術支持中心是江蘇省發展改革委員會的立項單位，江蘇省教育廳指定軟體人才實習基地。中心的建立，可以為江蘇 Linux 產業國際合作提供優秀的平台；可以利用軟體出口業務為江蘇 Linux 企業商業發展模式提供良好的借鑒；有助於大力發展和提高 Linux 人才教育培訓水平；能夠為中國 Linux 高新技術研究提供技術合作平台。建成後的江蘇 Linux 公共技術支持中心將是中國 Linux 產業的標誌和最有價值的 Linux 品牌。江蘇 Linux 公共技術支持中心的建立，將會全面整合江蘇及全國的 Linux 產業的優勢資源，更會是中國在作業系統領域和微軟展開競爭的 Linux 中心基地。

◎ 南京的 IT 研發優勢

江蘇 Linux 公共技術支持中心主任趙剛表示：「很多著名的 IT 廠商都在南京設立研發中心，包括 Intel、IBM、華惠、中興、東軟及中軟等等，新一代的網路通訊設備及智能手機都有利用 Linux 研發，而且中國已經有自己研發的龍芯處理器，配合自主研發的華鐳作業系統 RAYS LX 及永中 Office 辦公軟體，我們可以創造自主的軟件產業，以江蘇省校校通工程為例，當中有 14 萬台電腦全面裝設新華 Linux 華鐳作業系統桌面，成為全國最大的 Linux 成功案例。中國政府在推廣 Linux 應用的力道很強，但是單靠政府強制使用還是不足夠，市場需要更多的資訊、解決方案及人才，而且 Linux 及開放源碼產業也要有商業模式支持，江蘇省特別看好嵌入式市場的開發，因為我們有良好的電子工業，根據政府的數據，2006 年江蘇省軟體產業總值達 600 億元人民幣，其中 300 億元屬於嵌入式軟體，估計到 2010 年江蘇省軟體產業總值會提升至1,200 億元人民幣，其中 800 億元將屬於嵌入式軟體，現時有 50% 的嵌入式軟體是在 Linux 上研發，所以全省對 Linux 人才需求殷切。」

◎ 嵌入式 Linux 研發課程

根據資料顯示，目前大陸熟練的 Linux 作業系統開發人員只有 3,000 人左右，而 5 年內大陸 Linux 開發人才的需求將達 120 萬。趙剛指出：「過去大陸大部分高校都沒有開設 Linux 相關課程，Linux 專業教師數量也嚴重不足，並缺乏 Linux 教學實驗環境。為了增加有嵌入式 Linux 工程師的質量，我們聘請多位資深研發專家，獨家推出嵌入式 Linux 研發工程師課程。在中國，一個只有半年開發經驗的 Linux 人才，其月薪也可達 3,000 元（人民幣）以上，一個有一年以上開發經驗的 Linux 人才最低月薪可達 4,000 至 5,000 元，懂得嵌入式 Linux 研發的人才年薪可輕鬆超過 10 萬，所以有關課程很受學生歡迎。」

「中心已經精心打造擁有專業 ARM9 開發平台的嵌入式實驗室，學生可獲一人一套開發平台，課程內容包括隨堂實驗及 3 周項目實訓，透過整合知識，運用到企業真實軟體發展項目中，真正提高學員的實踐技能。中心更與南京多家著名 Linux 研發企業合作，學員入學即簽訂就業協議，畢業合格後保障就業。3 年來，已有千多名學員成功就業，薪資水平均在 3,000 元以上。目前中國已在大陸 42 所高校啟動 Linux 人才培養計畫，以平均每所有 800 名學生計算，目標對象有 33,600 人，現時我們正與江蘇省設有計算機課程的高校合作推廣上述課程，假以時日，相信可培育大量優質嵌入式 Linux 研發工程師。」

◎ 中國工程師關注 Linux 標準制定

根據市場研究機構預測，從 2007 年到 2011 年中國 Linux 市場複合增長率將達 21%，其中行業嵌入式系統更是一大商機，隨著行業資訊化的深入，不少行業用戶對嵌入式 Linux 作業系統的需求增加，例如在醫療行業中，Windows 作業系統的大部分功能並不實用，但這些多餘的功能佔用了很多系統資源，同時還給系統穩定帶來了風險。而證券業也可利用嵌入式 Linux 系統向股民提供股市行情。對於嵌入系統開發者而言，Linux 具有更佳的適應性、開放性和低成本的優點。在一項有關「中國電子工程師設計能力」的調查中，參與者可以在列出的 11 項中國已出台或正在制訂的自主標準中，選擇不超過 3 項自己關注的標準。結果 Linux 標準與 IPv6 標準雙雙以 22% 的關注度名列「中國標準關注度」的第 3 位，僅次於 DMB-T/H(29%) 和 TD-SCDMA (24%) 兩種標準，足證 Linux 受到廣大中國工程師的關注。
