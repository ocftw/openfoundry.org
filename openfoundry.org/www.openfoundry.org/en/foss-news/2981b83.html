Gartner Linux 短期內仍無法威脅 Windows 地位

Created at Friday, 29 September 2006 08:00      Last Updated on Friday, 09 March 2007 01:42 Written by 謝良奇
Gartner Open Source 大會上 Gartner 分析師 George Weiss 表示，未來 5 年內 Linux 在伺服器市場仍無法對 Windows 造成絕對的威脅。Linux 將逐漸達到與 Unix 同樣成熟的地步，虛擬技術 (virtualization) 則會是市場關鍵競爭元素。

在題為「企業 Linux 已經到來了嗎？」(Enterprise Linux: Has it Arrived?) 的議程中，Weiss 認為到了 2011 年，在全球伺服器作業 系統市場上，Linux 儘管會有大幅的成長，其營收仍然居於 Windows 與 Unix 之下。屆時 Linux 從 2006 年預期的 83 億，大幅成長到 120 億美元。Unix 預估有 159 億，Windows 則為 225 億。 雖然 Unix 市佔率仍高，但 Weiss 認為隨著 Sun 的 OpenSolaris 提供 Linux 之外的其它選項，到了 2011 年新的 Unix 實作會逐漸 減少。 Linux 隨著 90% 的核心貢獻來自於 Linux 廠商與散佈商，Linux 會 走向成熟，最終成為與 Unix 同等級的成熟產品。 由於 Xen 將成為 Linux 奠定在資料中心地位的關鍵推手，虛擬技術 (virtualization) 會成為市場的重要競爭利器。雖然 Xen 有著價格 與系統廠商支援等優勢，然而缺少管理工具、難與獨立軟體廠商授權 或定價模式接軌等，仍是 Xen 的弱點。微軟也不會坐視，並且已經在研發自家的 hypervisor 技術 Viridian。 紅帽 (Red Hat) 儘管成長迅速，但 Weiss 不認為該公司會成為 Linux 界的微軟。在美國市場紅帽享有高市佔率，但正面臨 Novell SUSE Linux 的挑戰。只要 Novell 適時調整腳步，很有機會在市場上取得成功。 此外，微軟的共享源碼 (Shared Source) 與提升產品與開放源碼軟體互通性等做法，也對紅帽構成了越來越大的市場競爭壓力。 Weiss 表示，未來面對 Linux/Unix、Windows 選擇時，決定因素將 取決於整體解決方案成本、管理工具、技術與可重用性。
