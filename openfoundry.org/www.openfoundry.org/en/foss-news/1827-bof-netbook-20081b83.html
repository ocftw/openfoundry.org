11/17 台灣社群企業 BoF-Netbook (Aspire One)

Created at Tuesday, 11 November 2008 00:05      Last Updated on Friday, 18 February 2011 02:04 Written by 呂健宇

◎ 活動時間：2008年11月17日(一)16:00-18:00
◎ 活動地點：台北科技大學 創新育成中心 3 樓 305 會議室 (台北市忠孝東路三段48號)
◎ 活動對象：歡迎社群朋友與 Netbook 同好來參加！

筆電 (Netbook) 已成為目前最夯的 IT 商品，是有助於推廣自由軟體應用的最佳媒介。自由軟體鑄造場 (OSSF) 秉持著促進『台灣社群與業界的交流互動』，邀請 Acer 團隊以同好聚會 (BoF) 方式對 Netbook 之軟體開發或應用進行討論與分享，歡迎有興趣的社群朋友們共同參與！

暫定話題：

●  Enhance Browser compatibility
●  GTK widget libraries & UI development tools
●  Enhance Open Office
●  Wrap open-source software for Aspire one

進行方式：

●  Acer 團隊的 Aspire One 軟硬甘苦談
●  社群自我介紹(以社群為單位)
● Netbook 同好交流

◎ 活動費用：免費 (現場約可容納15位，提供餐點與茶水)

◎ 報名方式：https://swan.iis.sinica.edu.tw/signupJ/signup.php?reg_id=20
◎ 特別感謝：智新資通股份有限公司提供場地
