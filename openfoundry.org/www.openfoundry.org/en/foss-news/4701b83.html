英飛凌、三星共推 UMTS_EDGE 雙模參考設計平台

Created at Friday, 04 February 2005 08:00      Last Updated on Friday, 09 March 2007 00:05 Written by 謝良奇
英飛凌（Infineon）、三星（Samsung）、Trolltech 與 Emuzed 將 共推全球第一套採用 Linux 作業系統、支援 UMTS/EDGE 的雙模智慧 型手機參考設計。此一整合軟硬體之平台，將可作為智慧型手機製造 商的 3G 產品藍圖之用。 此一參考平台於本月 14 至 17 日在法國 3GSM 大會上展出，屆時預 計展示尖端的 3G 及多媒體服務，如 UMTS/EDGE 雙模語音通訊、視 訊通訊、影像串流、高效能多媒體服務（包括 H.264, MPEG-4, AAC+, MP3）、高速網路瀏覽、3D 的 Java 遊戲等。

展示中將包含於高效能 Java 實作上的新 Java 應用軟體，以及 ARM 與 Superscape 的 Swerve 3D 繪圖引擎（rendering engine）、Opera 網路瀏覽器與 QSound Labs 的 3D 音效與來電鈴聲解決方案等。

英飛凌等四家公司將有意迅速跨足採用 Linux 之 UMTS/EDGE 智慧型 手機市場的手機製造商，視為該軟硬體參考設計的主要客戶群。 Trolltech 執行長 Haavard Nord 表示，該參考設計包括了軟硬體元 件與技術，可作為 3G 智慧型手機的產品藍圖。

EDGE 為歐洲與美洲佈展用的 2.75G 網際網路存取技術，UMTS 則是 3G 的網路存取技術，目前被應用於日本、歐洲等地。由於該平台支 援 UMTS 及 EDGE 標準，該平台將可立即運用普遍的行動網路架構， 同時提供寬頻多媒體功能。

在硬體方面，該平台採用英飛凌開發的 3G 智慧型手機參考板 （reference board），及三星的 S3C24A0 應用軟體處理器與相機模 組，此外尚含英飛凌的 MP-U UMTS/EDGE 數據機平台。英飛凌表示， 該參考板鎖定的是小尺寸的手機設計，並支援滑蓋（slider case） 設計。

該平台軟體平台為已預先整合妥善的解決方案，採用了來自英飛凌的 UMTS/EDGE 雙模協定堆疊、Samsung 最佳化後的 Linux 核心與驅動 程式，以及 Trolltech 的 Qtopia 圖形應用軟體平台和 Emuzed 的 影音軟體。

三星與英飛凌共同聲明中指出，開放源碼的 Linux 迅速成為智慧型 手機的軟體平台首選，由於其廣大開發社群，Linux 提供廠商相當大 的獨立性，以及使用者介面與品牌操作力。此一平台可讓製造商輕易 利用 Linux 所帶來的效率、彈性與功能性等優點。
