2009/4/22 自由軟體應用程式開發技術交流座談會 - MySQL

Created at Tuesday, 14 April 2009 19:33      Last Updated on Tuesday, 11 January 2011 22:01 Written by 呂健宇

MySQL是一個多人多工，並且基於 multithread 方式設計的資料庫軟體，它的執行效能相當高，並且十分穩定。目前 MySQL 被廣泛地應用在 Internet 上的中小型網站中。由於其體積小、速度快、總體擁有成本低，尤其是開放源始碼這一特點，許多中小型網站為了降低網站總體擁有成本而選擇了 MySQL 作為網站資料庫。


本次研討會將邀請貢獻 MySQL 開放源始碼 的 Sun Microsystems 技術顧問吳宏彬先生，介紹 MySQL 開放性技術以及 Sun 對於 MySQL 所提供的解決方案。

●  活動時間：2009年4月22日 下午14:00-17:00
●  活動地點：科技服務大樓(台北市松山區民生東路四段133號8樓) 808會議室
●  報名網址：https://www.oss.org.tw/seminar/meeting4/
