Adobe 加入 Linux 基金會 順勢推出 AIR For Linux alpha 版本

Created at Thursday, 03 April 2008 08:00      Written by 謝良奇/編譯
非營利組織 Linux 基金會 (Linux Foundation) 宣佈 Adobe Systems 加入的消息。基於該基金會專責加速 Linux 成長的目標，Adobe 將與其合作，塑造 Linux 成為豐富性網路應用軟體 (RIA) 與 Web 2.0 技術的領導平台。

Adobe 同時宣佈將 Adobe 整合執行時期環境 (Adobe Integrated Runtime，AIR) 的釋出前 alpha 版本，釋出給 Linux 社群。 這意味著，儘管 Adobe 大多數產品仍是私有軟體，該公司還是可以透過各種途徑，與開放源碼社群保持友好關係。

Linux 基金會執行董事 Jim Zemlin 表示，Adobe 加入基金會的決定，可說是其支持開放標準與開放源碼宣告下的自然舉動，藉此顯示該公司在軟體產業的領導地位與遠見。Linux 基金會贊助 Linux 核心的持續發展，並大力鼓吹 Linux 與開放源碼軟體。Zemlin 認為，Adobe 入會有助於增加 Linux 上的應用軟體開發，特別是 Web 2.0 應用軟體。

Adobe 技術長 Kevin Lynch 曾強調增加對開放源碼與標準貢獻的意願。Flex 開發框架的軟體開發套件，有助資料集中式 RIA 開發的 BlazeDS 技術，以及 ECMAScript 的開放源碼實作 Tamarin，都是 Adobe 過去對開放源碼界所作的貢獻。Adobe 並已加入 SQLite 協會 (SQLite Consortium)，同時參與各項開放源碼研討會。

Linux 基金會的年度協同大會 (Annual Collaboration Summit) 上，即將見到 Adobe 加入其他會員的行列。2007 年的協同大會上，Linux 社群多達 230 家領導廠商為了加速協同合作而齊聚一堂。

今年稍早 Adobe 已為 Windows 與 Mac 作業系統，推出 AIR 產品化版本。加入 Linux 基金會之餘，Adobe 也宣告推出釋出前 alpha 版本 AIR 給 Linux 社群。今年稍晚將可望看見最終版本推出。此外，Adobe 也釋出 Adobe Flex Builder 3 的 Linux alpha 版本。

Adobe 標準與開放源碼主任 David McAllister 表示，Adobe 為 Linux 用戶帶來關鍵 RIA 技術，例如 Adobe Flash Player 以及 Adobe AIR，可供在瀏覽器與桌面部署 RIA。Linux 基金會是極具價值的資源，提供可與社群共同合作的論壇，以確保 Adobe RIA 技術和 Linux 軟體平台的相容性。

目前應用軟體開發正逐漸朝網際網路發展。正式因為 Linux 開發社群提供了活躍的協同環境，軟體開發者得以擴展新網路技術與機會的創新，Linux 成為 Amazon、Facebook、Google 等企業的平台選擇。

運用 Flash、Flex、HTML 與 Ajax 建構 RIA 的跨作業系統執行環境 AIR，透過 Linux 版本的問世，RIA 開發者可將應用軟體推向 Linux 用戶，而無須為特殊平台撰寫額外程式碼。

自由開發者 Jonathan Snook 認為，快速與輕鬆建構相容 Windows、Mac 與 Linux 的應用軟體，是相當具威力的功能。過去開發者必須使用 Java 這些複雜的環境達成此目標，如今運用 Flash 或 JavaScript 即可建構桌面應用軟體，會是業餘開發者一大福音。
