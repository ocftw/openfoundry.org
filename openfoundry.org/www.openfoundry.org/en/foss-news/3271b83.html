Linux TCO 研究當道 Novell 鼓勵用戶多方參閱

Created at Friday, 07 October 2005 08:00      Last Updated on Friday, 09 March 2007 01:25 Written by 謝良奇
近來積極投身 Linux 陣營的 Novell，建議正在評估 Linux 的 Windows 與 Unix 用戶，應當參考各方的總持有成本（Total Cost of Ownership, TCO）研究，甚至是微軟贊助所做出的獲得事實（Get the facts）報告。一份 IBM 日前贊助研究的報告指出，Linux 在部署及 管理上的成本皆優於 Windows。

Novell 解決方案管理經理 Brian Green 近來鼓勵用戶藉由包括微軟 在內的各方總持有成本研究，作為決策考量依據。Green 在日前倫敦 的 LinuxWorld 上表示，所有的這類報告多少都有可資依據之處， Green 認為，並非用戶閱讀到的所有資訊都是支持 Linux，決定權依舊在用戶自己身上。Green 也建議有關總持有成本的最終決定，應該 依組織個別的情況來考量。 當前全球最大 Linux 散佈者 IBM，日前發佈了一份委託 Robert Frances Group 所作的研究報告，報告中指出 Linux 的總持有成本 優於 Windows。 IBM 資深副總裁兼集團主管 Steve Mills 指出，根據此份報告多項獨立研究皆顯示 Linux 於一般工作負載下的成本優勢。 IBM 此一針對 20 家中大型企業 IT 主管所作的報告宣稱，購買、實作與運行採用 Linux 的 x86 伺服器作為應用伺服器，要比使用 Windows 的同樣伺服器便宜 40%。一般認為，IBM 此份報告主要是針對微軟所作的面對事實廣告而來。微軟贊助分析師發表的報告中指出，在總持有成本上，微軟的 Windows 要優於 Linux。 儘管廠商贊助研究報告並不罕見，但是在懷疑微軟是否重施行銷恐怖 心理戰（FUD）的陰霾籠罩下，此一調查報告著實帶來外界眾多關切的眼光。微軟平台策略經理 Martin Taylor 為此辯護，他表示，這 類產業分析本來就需要贊助才能進行研究。 Robert Frances 並非首次著手 IBM 相關的研究報告，過去便曾針對 Linux 以及 IBM Unix 的競爭對手 Sun Microsystems，發表過比較 Solaris 與 Linux 的研究報告。
