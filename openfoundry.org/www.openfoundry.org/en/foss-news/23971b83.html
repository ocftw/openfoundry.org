Yahoo! 釋出 Hadoop 安全增強和工作流程管理工具

Created at Thursday, 01 July 2010 08:00      Last Updated on Monday, 12 July 2010 00:46 Written by 謝良奇/編譯

Apache Hadoop 的主要開發者 Yahoo!，宣佈釋出 Hadoop with Security、Oozie，以進一步加強 Hadoop 的功能，新釋出的兩套開放源碼軟體可為企業資料賦予價值，達到更好的企業管理，可望加速 Hadoop 在主流企業中的普及。Hadoop 是位於大規模資料與雲端運算核心的開放源碼技術。

當網際網路使用成長，資料無所不在，針對如何以安全而有效的方式管理資料，使各類型企業面臨了挑戰。作為全球最大線上公司之一的 Yahoo!，將 Hadoop 打造為一企業級平台，並在其內部各處加以應用，開發個人化的使用者體驗。Hadoop 在 Yahoo! 知名的全球網站，例如 Yahoo! Search、Yahoo! Mail，扮演了關鍵的角色。

Yahoo! 執行副總裁兼產品長 Blake Irving 表示，Hadoop 是科學與大量資料的交會之處，也是支撐該公司創新用戶與廣告商產品的技術基礎。Yahoo! 的雲端和 Hadoop 讓該公司得以迅速個人化其內容與廣告。

在 Yahoo! 主辦的第 3 屆 Hadoop 大會 (Hadoop Summit) 上，該公司宣佈了 Hadoop with Security 和 Oozie 的 beta 釋出。由於這些工具提供對於企業敏感資料的更有效管理，並可在 Hadoop 上實現複雜程序的執行，企業將因為這些開放源碼的釋出而受惠。Hadoop with Security 和 Oozie 可相互運作，並且在 Yahoo! 的上萬部伺服器經過測試與佈署。

Hadoop with Security 對 Hadoop 提出一系列重要的安全更新，透過整合 Hadoop 和成熟的開放源碼驗證標準 Kerberos，可實現更安全的協同合作和經驗證資料的分享。藉由該工具，將可達成服務多重租賃 (multi-tenancy)，或者讓多個內部單位共享硬體，提供經驗證的安全存取並處理敏感資料。

 Yahoo! 的 Hadoop 工作流程引擎 Oozie，是開放源碼的工作流程管理和協同合作引擎，用來管理 Hadoop 上的執行工作，包括 Hadoop Distributed File System、Pig、MapReduce。這套工具是針對 Yahoo! 需要管理複雜工作程序，並在全球規模上進行資料擷取、轉換、載入 (ETL) 的嚴苛使用情況而設計。

IDC Research 研究主任 Melanie Posey 指出，各領域的企業都在找尋方法，從他們所收集的大規模資料中提取價值，Apache Hadoop 正是在此規模上進行資料處理的有效解決方案。Hadoop 已經成熟，隨著 Kerberos 驗證的加入更成為適於企業的雲端運算技術。現在各種規模的組織都可以在自己的系統上，運用 Yahoo! 的 Hadoop 投資與佈署，而無須自行從頭開始。
