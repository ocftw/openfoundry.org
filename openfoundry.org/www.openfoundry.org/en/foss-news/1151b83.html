英、法議會紛投入開放源碼懷抱

Created at Thursday, 11 January 2007 04:50      Last Updated on Thursday, 08 March 2007 05:42 Written by webmaster

歐洲政府單位近來不約而同擁抱開放源碼。英國伯明罕 (Birmingham) 議會剛結束長達一年的 Linux 桌面試驗，宣佈計劃成功並將持續增加開放源碼軟體的使用。法國議會則計劃在明年 6 月開始，使用執行 Linux、Firefox 瀏覽器與 OpenOffice.org 的桌面電腦與伺服器。

根據 Maastricht 大學的報告指出，與其它歐盟國家相比，英國在開放源碼軟體使用的腳步上較為落後。只有 32.1% 的英國地方政府在使用開放源碼軟體，而歐洲國家平均則為 78.7%。 英國伯明罕日前認為其長達一年的桌面 Linux 實驗結果成功，這項由開放源碼協會 (Open Source Academy) 執行，計劃在該市議會圖書館服務上安裝 330 台 Linux 桌面的計劃，將可供開放源碼協會往後提供給其它有意採用開放源碼軟體的公領域機構。 報告指出該實驗花費成本要高於安裝 Windows 系統，對此，伯明罕市議會 IT 主管 Glyn Evans 認為，試驗與熟悉新技術是成本較高的原因，此外與現有複雜系統整合也是原因之一。此外，報告指稱藉由安裝 Windows XP 系統可省下 10 萬 5 千英磅成本，也因為採用了微軟提供給議會的特別折扣授權費來計算而引起爭議，部份人士批評這是迫使公領域機構擁抱開放源碼技術的手段。 儘管 Evans 也承認計劃起初野心過於龐大，預計在一年實驗期內在員工與公用電腦安裝 Linux。事實上該計劃仍未安裝 Linux 桌面系統，只在公用與員工電腦上執行 OpenOffice.org 與 Firefox 等開放源碼軟體。根據 Evans 的說法，該議會仍計劃將這些桌面電腦改用 Suse Professional 9.3。伯明罕更希望大幅增加參與此一計劃的桌面電腦數。 與英國相隔英吉利海峽的法國，僅管只有在伺服器上，但不少政府單位早已是 Linux 的用戶。日前法國議會更進一步表示明年 6 月開始， 將使用執行 Linux 的桌面與伺服器。法國議會在日常文件上也將使用 OpenOffice.org。預計安裝 Linux 的桌面電腦將達 1,154 台。 據報導法國議會決定採用開放源碼軟體，主要是著眼於可觀的成本節省。這也是法國政府擁抱 Linux 桌面的最大動作。 研究組織 NPD 的軟體產業分析主管 Chris Swenson 認為把改用 Linux 視為節省成本的代名詞，忽略許多公司部署 Linux 時在訓練與支援上所花費的成本。再者，法國議會並未公佈採用的 Linux 散佈套件，Swenson 認為這一點相當重要。他表示如果向紅帽 (Red Hat) 等 Linux 廠商購買軟體，仍需支付授權、支援與維護費用。
