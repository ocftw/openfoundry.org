Ubuntu 創辦人遊說亞洲政府單位推廣開放源碼技術

建立日期 2006-02-10 08:00      最近更新在 2007-03-08 08:23 作者是 謝良奇
Ubuntu Linux 的創辦人 Mark Shuttleworth 日前到亞洲各國，希望遊說政府多加提倡一般使用者使用開放源碼技術。 在菲律賓一站，Shuttleworth 表示政府應在一般用戶間廣推開放源碼平台，做為家庭計算應用的新選擇。他強調開放源碼應用軟體可以立即提供給需要成本低廉選擇的客戶。由於許多學校相關應用軟體都牽涉到客制化，教育機構也能從開放源碼當中受惠。

Shuttleworth 表示，讓用戶同時有私有及替用平台供選擇相當重要，政府與教育機構在這方面能有所幫助。

Shuttleworth 在菲律賓的行程與服務提供者及電腦製造商會面，討論在電腦中安裝 Ubuntu Linux 的事宜。Shuttleworth 也在馬尼拉舉辦 Ubuntu 作業系統研討會。

以使用者親和力著稱的 Ubuntu Linux 有廣大的用戶與開發者社群支援，同時也有最快的更新的周期，每六個月就推出新版本。最新版本 Ubuntu 6.04 預計會在四月推出。
