Mac 與 Linux 電腦可能無須安裝中國內容過濾軟體

建立日期 2009-07-09 08:00      作者是 謝良奇/編譯
中國日前要求國內外個人電腦製造商，在該國內銷售的所有電腦上，安裝綠壩 (Green Dam Youth Escort) 內容過濾軟體，用以過濾色情與敏感的線上政治內容。現在傳出，目前只有 Windows 個人電腦才需要安裝這套引發爭議的軟體。

儘管中國政府正式要求該國內銷售的所有電腦，都必須包含這套軟體，如今包括 Apple 電腦與安裝 Linux 的個人電腦在內，無法安裝綠壩軟體的系統，將不需要隨機附上該軟體。根據一匿名個人電腦廠商的說法，此一免除條款也適用於無法滿足該軟體安裝所需技術要求的電腦系統。

這包括了執行 Windows 64 位元版本的許多中高階個人電腦，例如因為 32 位元程式碼較低的 4GB 記憶體上限，而傾向使用 64 位元版本的多數 Windows 7 個人電腦。

根據 PC World 的消息，因為綠壩軟體並無 Mac OS X 版本，在北京 Apple Store 銷售的 Mac 電腦將不會安裝該軟體。然而，一旦 Mac 版本完成後，Apple 將依照規定，在系統中預先安裝綠壩軟體。

一位 Lenovo 發言人表示，中國工業及資訊技術部 (Ministry of Industry and Information and Technology) 並未要求非 Windows 系統必須包含該軟體，這表示安裝 Linux 的 Lenovo 電腦，將不會安裝任何綠壩軟體。

容許以上電腦無須安裝綠壩軟體，將有效導致對該軟體有疑慮的用戶，轉向 Mac 與來自 Dell 或 Lenovo 等大型電腦製造商採用 Linux 的特定產品。根據官方的說法，該軟體只是用來過濾網際網路的色情內容以保護孩童，但預料也將用來阻擋挑戰大陸官方觀點的政治內容，更謠傳可能包含後門控制與監控功能，協助政府追蹤使用者活動。Sony 是首家自行提供該軟體的業者，日前已在產品中隨附此一軟體，但表示並不保證該軟體能執行。

目前中國政府為了緩和國際上的疑慮，延後此一規定的實施時間，並表示使用者不一定要安裝該軟體，一旦安裝後也可以合法地加以移除。

目前並不清楚綠壩軟體是否會釋出其他作業系統的版本。一名來自與綠壩軟體主要開發者 Jinhui Computer System Engineering 隸屬同一集團企業的服務代表透露，該公司正在測試非 Windows 平台上的綠壩軟體。但並不確定其中是否包括 Mac OS X。
