LINA 1.0 beta 釋出 允許 Linux 軟體跨越主要作業系統

建立日期 2009-10-15 08:00      作者是 謝良奇/編譯

Lina Software 宣佈該公司的應用軟體移植性解決方案 LINA 1.0，進入 beta 釋出階段。此一開創性技術可協助 Linux 軟體開發者更有效地進入新市場，簡化終端使用者體驗，在無須更換現有硬體情況下，執行 Linux 應用軟體。

LINA 是一套企業級的開放源碼軟體，可允許 Linux 應用軟體在 Windows、Mac 與 Linux 各主要作業系統上執行，並提供原生介面外觀。LINA 讓開發者使用開放源碼的 Linux 工具撰寫並編譯程式碼，然後快速地轉換所產生的應用軟體到 LINA 的泛用安裝程式上。

透過在各作業系統背景執行的 LINA 執行環境，使用者可以簡單地安裝並執行經 LINA 處理過的 Linux 應用軟體。

根據該公司表示，LINA 1.0 beta 版本已經透過 GNU General Public License 第 2 版授權釋出。這個版本包含了包括點選 (click-through) 式的應用軟體安裝程式，用來簡化企業部署並降低昂貴的終端用戶支援成本。

此一解決方案允許 Linux 應用軟體可以符合主作業系統 (host operating system) 的外觀與使用體驗。藉由 LINA 的單一程式碼基礎，開發者利用單一軟體即可銷售給三大主要作業系統上的用戶。

開發者如今可以重用為 LINA 編譯的開放源碼 Linux 元件，加速產品的上市時間。LINA 應用軟體在 Linux 作業系統上執行可達原生速度，在 Windows 與 Mac 作業系統上，也可達近乎原生應用軟體的速度。

Lina Software 執行長 James McGreen 表示，他們藉由完全不同與以往的商業模式，希望為 Linux 開放源碼軟體創造廣泛的市場接受度。他們公司的宗旨正抓住了此一革命性目標，同一世界，同一份源碼。
