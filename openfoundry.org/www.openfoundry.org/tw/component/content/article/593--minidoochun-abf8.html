嵌入式系統學習工具 Minidoochun 工作坊

建立日期 2005-03-03 08:00      最近更新在 2011-07-21 17:38 作者是 涂敏怡

光寶科技新技術暨產品研發中心資深經理曾瑞源，日前於二月二十三 日應自由軟體鑄造場之邀，至中央研究院資訊科學研究所演講，分享 如何用 Minidoochun（迷你豆漿）嵌入式系統學習套件，以進行嵌入 式 Linux 設計方法。


Minidoochun 是一套快速開發嵌入式系統 root file system（rootfs） 的模具組，也是從 Doochun Linux（豆漿 Linux）延伸出來的子計畫。 承襲豆漿 Linux 強調「簡單」的精神，其特點在於可以快速製作定 型化的 rootfs，而且修改非常容易，只需要在預鑄的 rootfs 放置 所需的應用程式以及函式庫，約十分鐘的時間就可以製作出全新的 rootfs。

曾瑞源表示，Minidoochun 非常適合做為 Embedded Linux 教學環境， 學習曲線比 uClinux 低，搭配 VMware 這套硬體平台模擬軟體，使 用者即可以獨立在 Windows XP 作業環境上運行及測試，快速練習各 式 Embedded Linux 的設定與應用程式開發。

Minidoochun 不是 Linux 散佈套件，而是一套工具組，採用 BusyBox 為基本架構，目前發展至 3.0 版，已經有 X86 版本以及 RISC 版本， 前者有內附 Linux Kernel，因此可以用來設計出完整的 Embedded Linux Distribution。

在實際應用方面，曾瑞源透露，雖然 Minidoochun 目前定位為學習 用途的開發工具，但是大陸一些廠商已經採用 Minidoochun 進行產 品的開發。未來 Minidoochun 除了加入除錯與 IDE 整合發展環境、 繼續使用標準 Kernel 外，未來也將整合 uClinux 以及如何彌補 uClinux 的不足方向邁進。

曾瑞源本身也是台灣自由軟體社群資深推廣者，約在十年前左右，曾 經寫過台灣最早的 Linux 技術書籍，所寫文件並在網路上自由流傳， 因而極受本土自由軟體社群的推崇，近年來曾瑞源的研究興趣轉移至 嵌入式系統教學與開發。
