Acer 推出預先安裝 Ubuntu 的筆計型電腦 Aspire 5710Z

建立日期 2007-07-20 16:00      最近更新在 2007-08-13 06:10 作者是 謝良奇
全球最大筆計型電腦製造商之一宏碁 (Acer) 日前在新加坡銷售 Aspire 5710Z，預先搭載了 Ubuntu Linux 而非 Windows，加上 Acer 方面先是表示這類型安裝 Linux 的筆計型電腦將不會在英國推出，事後卻自行推翻這個說法，Aspire 5710Z 意外地讓預先安裝 Linux 的筆電的市場需求成為關注焦點。

Ubuntu 是目前最受到使用者歡迎與最容易使用的 Linux 散佈套件之一。Acer 近日在新加坡開始銷售安裝 Ubuntu Linux 的 Aspire 5710Z 筆計型電腦。Acer 此一舉動與以往傳統個人電腦均搭載 Windows 不同。

然而一位 Acer 發言人表示，Acer 並無在英國銷售 Linux 電腦的計劃。因為英國對這類系統沒有市場需求，該公司目前不會提供英國使用者此類型的系統。該發言人並表示，假如有市場 Acer 當然會銷售這種產品，而 Acer 的英國經銷商會將這類需求傳達給 Acer。

目前有 Linux 產品線的其它主要製造商就是 Dell，從 5 月開始 Dell 就已經在美國供應這些產品。Ubuntu 主要贊助者 Mark Shuttleworth 也表示過，Dell 應該推出更多以開放源碼為基礎的筆計型電腦。Dell 對此建議未做評論。

Dell 的 Linux 系統也沒有在英國上市。但 Dell 表示過很快就會改變這個情況。

不過幾天後 Acer 對外表示，雖然現在 Acer 沒有在英國銷售 Linux 機種，但未來很可能會推出。加上一位新加坡經銷商的銷售人員表示，Acer 出貨的 Aspire 5710Z 根本就沒有安裝任何作業系統，只是一台陽春電腦 (naked PC)。

這位來自 Memory World 的銷售人員說，作業系統是他們自行安裝的，5710Z 只是一部入門機種，不適合安裝 Windows Vista，而選擇 Ubuntu 的理由是，Ubuntu 比起 XP 毫不遜色。

令外界摸不著頭緒的是，有其它 Aspire 5710Z 的經銷商表示，他們收到的機器已經預先安裝了 Linpus 而非 Ubuntu。但另一位 Memory World 的銷售人員卻說，Acer 給的機器的確有 Linux，但不是 Ubuntu。

在 Acer 銷售此款電腦前，只有 Dell 考慮過安裝以 Ubuntu 為基礎的產品。近來市場出現許多對於陽春電腦的需求，因為 Linux 用戶經常為了要花錢在他們用不到的 Windows 上而感到困擾。
