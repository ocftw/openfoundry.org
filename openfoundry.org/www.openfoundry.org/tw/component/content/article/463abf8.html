紅旗加入 OSDL 共同推動中國 Linux 發展

建立日期 2005-01-16 08:00      最近更新在 2008-05-02 22:12 作者是 謝良奇/編譯
開放源碼發展實驗室（Open Source Development Labs，OSDL）日前 宣佈，中國 Linux 公司紅旗（Red Flag）已加入 OSDL，將參與該組 織的桌面 Linux（Desktop Linux）電信級 Linux（Carrier Grade Linux），與資料中心 Linux（Data Center Linux）三個工作小組。 開放源碼發展實驗室為一全球性的產業協會，旨在推動 Linux於商業 領域的發展，其成員包括有 Intel、HP、Red Hat、Novell 等大廠。 紅旗是中國主要的 Linux 開發公司，銷售嵌入式產品、伺服器、桌 面系統與垂直市場的 Linux 堆疊。這家由國家支持運作的 Linux 軟 體開發業者，負責開放源碼於中國政府、教育、電信及金融領域的應 用推展。該公司連接 Oracle、AMD、Dell、HP 等公司以認可其產品， 並支持推動 Asianux 成為亞洲 Linux 標準。

紅旗執行總裁 Chris Zhao 針對加入 OSDL 一事表示，Linux 在中國 獲得很大的成功，有相當大潛力將 Linux 推展到各市場。紅旗加入 國際認可的 OSDL 組織，全球 Linux 社群將因紅旗對 Linux 與開放 源碼發展的貢獻而受惠。紅旗將連同 OSDL 與其成員，分享其經驗與 技術，進一步於中國、亞洲與全球推展 Linux。

根據 OSDL 的資料，中國是全球 Linux 成長最迅速的地區之一，中 國市場研究機構 CCID Consulting 預測未來五年內，將有 50% 的複 合年成長率。OSDL 更於去年八月，於北京設立辦事處。OSDL 執行長 Stuart Cohen 指出，OSDL 承諾將持續推動中國地區，這塊專家預估 全球最大軟體市場之處，未來的 Linux 發展。

成為 OSDL 的組織成員（Institutional Member），可以取得技術郵 件論壇、參加技術小組，獲得加入委員會、參與工作小組會議的權利。 取得會員資格最大的好處是可以獲得與 Linux 開發者、廠商和其它 會員互動的機會。
