Nokia 推出行動電話瀏覽器與開放源碼入口網站

建立日期 2005-11-04 08:00      最近更新在 2007-03-09 00:47 作者是 謝良奇
Nokia 日前釋出開放源碼行動 Web 瀏覽器，同時推出入口網站，分享該公司開放源碼參與計畫的資訊。

行動電話大廠 Nokia 不久前推出了新的入口網站，發佈該公司參與的開放源碼計畫相關資訊，其中包括了最近釋出供行動設備使用的 Web 瀏覽器。 今年 6 月 Nokia 宣佈開發可供其 S60 智慧型手機軟體平台使用的開放源碼瀏覽器，以改善行動電話用戶的網站瀏覽經驗。一般而言，未經針對行動電話最佳化的網站，很難在行動電話螢幕上瀏覽，網站必須針對行動電話加以特殊設計。 該瀏覽器日前已釋出給 S60 第三版的授權用戶。該瀏覽器採用蘋果電腦 Safari 瀏覽器的 WebCore、JavaScriptCore 為基礎。Safari 瀏覽器本身則基於來自 KDE Konquerer 瀏覽器的 KHTML 以及 JavaScript 引擎。 Nokia 表示，該新瀏覽器能將網頁，以網站開發者設計的原始風貌，顯示在行動電話用戶面前。此外，該瀏覽器還包括彈出視窗封鎖、RSS 閱讀及文字搜尋功能。 Nokia 推出作為提供其開放源碼專案資訊之用的入口網站上，除了該新瀏覽器的訊息，還包括 Nokia 用以發展 Linux 網路平板設備的 Maemo 開發平台，以及 Python 程式語言。
