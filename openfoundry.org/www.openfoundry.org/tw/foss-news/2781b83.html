Asianux 聯盟將推展亞洲版 Linux 散佈套件

建立日期 2006-04-20 08:00      最近更新在 2007-03-09 01:54 作者是 謝良奇
中國紅旗軟體（Red Flag Software）、日本 Miracle Linux 與南韓 Haansoft 計畫合資發展及銷售亞洲版本的 Linux 軟體產品。 紅旗行銷經理 Chase Fan 表示，名為 Asianux 的該合資計畫，將在24日的記者會上公開發表。有關合資結構細節，他則不願多加透露，只表示經過兩年研發，如今他們的產品與銷售管道已十分成熟。

這三家公司於 2004 年時宣佈共同開發 Asianux 散佈套件，該合作受到來自三個國家政府與產業的支持。Asianux 這套 Linux 伺服器作業系統鎖定亞洲企業用戶，目前已應用在南韓國家教育資訊系統與中國國家郵政系統上。 研究公司 Gartner 副總裁 Andrea Di Maio 認為 Asianux 標準被運用在大型政府專案，有助 Linux 在亞洲地區的成長。Asianux 公開原始碼及低成本等特性，將可挑戰微軟 Windows 平台。此外，Asianux 的競爭對手也包括歐美 Linux 廠商如 Red Hat 等。 近來在 Linux 事業獲利狀況頗佳的 Oracle，在 Asianux 這套亞洲 Linux 散佈套件背後也扮演了重要角色。根據 Miracle Linux 的說法，日本 Oracle 子公司握有 Miracle Linux 百分之 50.5 的股權。 紅旗、Miracle Linux 與 Haansoft 將以各自品牌，在中國、日本及南韓銷售以核心系統搭配區域化功能的 Asianux。
