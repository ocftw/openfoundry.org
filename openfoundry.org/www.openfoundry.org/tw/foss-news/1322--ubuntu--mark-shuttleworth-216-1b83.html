再掀 Ubuntu 旋風 Mark Shuttleworth 2/16 台大演講

建立日期 2007-11-27 22:52      最近更新在 2011-01-06 22:25 作者是 webmaster
◎ 公開演講時間：2006/2/16 (四) 17:30~19:00

◎ 公開演講地點：國立台灣大學 Room 102

（詳細地址請注意 https://www.openfoundry.org 公告）

◎ 公開演講題目：Ubuntu － Linux for Human Beings

Ubuntu Linux 背後 Canonical 公司的創辦人 Mark Shuttleworth（也是史上第二位太空旅客）於一月底至二月中旬率眾進行亞洲之旅，其中 2/16,2/17 兩天將安排在臺北訪問。



此行的目的主要是在亞洲推廣 Ubuntu Linux，特別是將它介紹給各國政府及硬體/系統廠商，同時也要與亞洲各地的 Ubuntu 社群會面。 Shuttleworth 已特別表達尋求臺灣硬體廠商合作的高度興趣，對於台灣廠商跨足印度及其他亞洲國家市場有不錯機會。



Ubuntu 是目前 Linux Distros 的一種，也是目前包括聯合國 UNDP-IOSN，以及目前許多非洲、歐洲的NGO（如 go-opensource.org ）與部份政府機構所使用的版本。在著名的自由軟體機構 OSDL 於 2005 年 11 月所作的調查顯示，目前使用或想要轉換使用 Ubuntu 的比例，高達 53%，是所有版本之冠（註四）。Ubuntu 在國外已經興起一陣使用熱潮，這次 Mark Shuttleworth 來台為 Ubuntu 宣傳，可望再

度掀起 Ubuntu 旋風！



本次 Ubuntu 亞洲之旅來台灣的代表為：

* Mark Shuttleworth - Founder

* Hande Bayraktar - Business development (and internationalisation)

* Sarah Haskey (Tour Mum)

* Malcolm Yates - Alliances and Partners
