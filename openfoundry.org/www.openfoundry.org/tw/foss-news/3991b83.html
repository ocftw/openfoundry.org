OpenBSD 財務吃緊 2007 黑客松可能停辦

建立日期 2006-03-24 08:00      最近更新在 2007-03-09 00:37 作者是 謝良奇
OpenBSD 以其在安全性與易用性上的卓越表現，在新作業系統層出不窮的今天，仍是許多技術玩家最愛的平台。然而，OpenBSD 正面臨財務困難的窘境，直接衝擊到 OpenBSD 未來的發展。 在 OpenBSD 3.9 版 CD 訂購起跑同時，OpenBSD 傳出財務吃緊的消息。OpenBSD 在過去 2 年中分別虧損了 2 萬美元，對 OpenBSD 這樣一個本來就沒有贊助單位的專案而言，累積下來的財務困難已經危及到專案的未來的開發。

最直接的影響就是 2007 的黑客松 (hackathon) 有可能因為財務狀況不允許而取消，對 OpenBSD 與 OpenSSH 的開發將產生嚴重的衝擊。

來自世界各地的作業系統開發者會在黑客松時，齊聚一堂花上一週的時間進行腦力激盪與撰寫程式碼，由於開發者得以彼此貼近討論，許多重要的 OpenBSD架構與未來版本的開發都是在黑客松時完成。大型黑客松大約要花上 3 萬美元，以單一主題為主軸的小型黑客松約需1 萬美元。

和其它專案不同，OpenBSD 幾乎沒有組織贊助，這一點跟該專案強烈的理想主義有很大的關係。OpenBSD 相當強調正確性與單純性，任何 沒有同時更新文件內容的程式碼修訂，都無法進入 OpenBSD 原始碼樹中。因此其手冊永遠處於正確無誤的狀態。在授權上也是一樣，其授權極力避免加入需要律師才能解讀的文字。

反映在該專案的獨立性上，OpenBSD 要求所有捐款都必須是無附代條件。不幸的是，這種要求大多不被贊助單位所接受。也因此，即便在許多商業產品如路由器或作業系統中都可見 OpenBSD 的影子，OpenBSD 團隊開發的 OpenSSH 也可見於 Sun Solaris、Mac OS X 及多數 Linux 散佈套件，卻少有組織願意回饋 OpenBSD。

OpenBSD 經費主要來自於 OpenBSD CD 的銷售，希望鼓勵用戶購買 CD，該專案並不提供常見的 CD 映像檔下載服務。OpenBSD 用戶只能選擇購買 CD 或透過 FTP 進行安裝。然而 FTP 安裝的比率逐年上升，越來越少用戶選擇購買 CD，CD 銷售率衰退可說是造成 OpenBSD 專案近年虧損的主因。

如何補救財務上的窘境，除了呼籲用戶儘可能透過訂購 CD 取得 OpenBSD 3.9 之外，OpenBSD 專案也希望使用 OpenBSD、OpenSSH 的各大專案能伸出援手，讓 OpenBSD 與 OpenSSH 持續運作下去。
