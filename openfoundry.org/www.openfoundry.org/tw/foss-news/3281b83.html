Linux 上的下一代 Centrino 技術支援

建立日期 2004-03-07 08:00      最近更新在 2007-03-09 01:24 作者是 謝良奇
加拿大軟體公司 Linuxant 日前釋出 DriverLoader 1.6 版，支援 Intel 下一代具 54mbps 802.11g 技術的 PRO/Wireless 2200BG 卡，以及 Centrino 系統的 Wi-Fi Protected Access (WPA) 安全標準。 Intel 是否會針對 Centrino 技術提供 Linux 支援，有各種不同的說法。Intel 發言人 Scott McLaughlin 表示，Intel 正在致力於 Centrino 的 Linux 支援，但尚未決定釋出的時間與方式。過去在 Linux 上要使用 WLAN，多半得尋求 third party 提供的 Linux 驅 動程式。

Linuxant 的 DriverLoader 是一套相容包裝軟體（compatibility-wrapper）。透過 DriverLoader，便可在 Linux x86 系統上，使用硬體廠商提供的標準 Windows NDIS 驅動程式。許多用戶不耐等待 Intel 釋出原生 Linux 驅動程式，轉而安裝 DriverLoader，以便於 Linux 環境下，使用 Centrino 無線區網設備。

DriverLoader 最新版已可由 Linuxant 網站下載，新版提升了對 Intel PRO/Wireless、Broadcom、Intersil Prism GT/Duette/Indigo 等多種 Windows 無線區網驅動程式的相容性與穩定性。

Pontus Fuchs 也提供了一套採 GPL 的 ndiswrapper，不過目前該軟體尚未能支援下一代的 Centrino。
