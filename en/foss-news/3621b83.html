新版 Opera 支援嵌入式系統 加入 IRC、RSS 等功能

Created at Tuesday, 25 May 2004 08:00      Last Updated on Friday, 09 March 2007 01:00 Written by 謝良奇
Opera Software 為該公司新款 Opera 瀏覽器加入 IRC，以及使用者介面改善。新版 Opera 在不大幅提高執行記憶體、軟體容量需求的前提下，如今除了瀏覽器，更包含了電郵軟體、新聞閱覽器（newsreader）、IRC、整合式 Google 搜尋等功能。新的 Opera 支援 Linux、嵌入式 Linux，以及其他普遍的嵌入式和一般平台。

Opera 是一款跨平台的網路瀏覽器，本屬於挪威電信公司 Telenor 的研究專案，於 1995 年獨立為 Opera Software ASA 發展公司。Opera Software 所開發的 Opera，是針對多種平台、作業系統及嵌入式網路產品的高品質瀏覽器產品。Opera 表示，Opera 7.50 包含了功能強大的電郵軟體。此一電郵軟體除了現在電郵軟體應該有的功能，像是過濾器、垃圾郵件管理、連絡人資料庫、文字搜尋、拼字檢查外，也身具 NewsGroups 和 RSS feeds 的處理能力。Opera 可以免費使用，但在未註冊的情況下，會顯示 Google Ad-Words 的廣告區。註冊費為 40 塊美金。雖然可以用 CSS 技巧擋掉廣告，不過因為 Opera 的免費版本有賴廣告收入，因此網友建議程式高手不要寫出廣告阻擋的 plugin。Opera 的外觀可自訂，使用者從 Opera 的選單中下載新的外觀。由於採用了 QT API，Opera 的執行速度頗佳，更可能因此讓 Opera 有了優於 Mozilla/FireFox 的 Bitstream 字形呈現效果。Opera 提供針對中小型螢幕施以最佳化的版本，同時 Opera 也是新款 Zaurus PDA 的預設瀏覽器。Opera 尚為嵌入式設備納入了一項特殊功能，稱為 Small-Screen Rendering（SSR）。該項技術可調整網頁至，適合 PDA、手機等裝置最佳的瀏覽狀態。Opera 也包含了對 DHTML 及 WAP 的支援。
