Ubuntu Linux 8.10 桌面、伺服器釋出 各自加強 3G 連線、Java 部署與虛擬化技術

Created at Tuesday, 28 October 2008 08:00      Written by 謝良奇/編譯

Canonical 宣佈 Ubuntu Linux 桌面與伺服器作業系統的最新版本，各自推出正式釋出版本。鑑於小型行動個人電腦的興起，新版 Ubuntu Linux 桌面系統加強 3G 網路連線能力，至於伺服器版本則在 Java 部署、系統管理與虛擬化技術等方面，提供更好的支援能力。

此次更新的 8.10 桌面作業系統為使用者新增了一系列可用性功能。Ubuntu 8.10 具備透過筆記型電腦 USB 外接裝置、手機或藍芽 (Bluetooth) 連線，經由 3G 數據機 (modem)，偵測並連線至 3G 網路的能力。這項功能將使 Ubuntu 更加適於作為 Netbooks 與其他超可攜電腦上的作業系統。配備小於 10 吋以下螢幕的低價筆記型電腦 Netbooks，近來成為筆電市場中炙手可熱的區域。

Canonical 這回讓使用者能將 Ubuntu 隨身攜帶。新版 Ubuntu 加入了應用軟體，可以讓使用者將 Ubuntu 寫入 USB 磁碟中，以便於在其他電腦上使用 Ubuntu。此外，這一項功能也提供使用者燒錄光碟之外，另一個更快速也更為簡便的選項。

新的 Ubuntu 另一項主要功能，是讓使用者快速建立會客時段 (guest session)，便於將電腦借給其他使用者使用，而不會打亂現有程式或設定，作業系統也會保護使用者資料。

Canonical 這次為新版 Ubuntu，整合了 Gnome 2.24 桌面環境，其中增加了新的即時傳訊軟體、內建時間追蹤器 (time tracker)，以及最新  Ekiga 3.0 影音會議軟體，同時改善了檔案管理功能。

Gnome 基金會 (Gnome foundation) 執行董事 Stormy Peters 表示，Ubuntu 8.10 藉由與 Gnome 等開放源碼技術合作，為使用者建立了無縫連接無線與行動通訊網路 (cellular network) 而隨時上線的系統。Peters 說，使用者無須在為找尋網路而擔心，他們的電腦一直是上線中。這對行動與桌面運算來說是一大突破。

另一項有趣的功能是 Ubuntu 整合來自 BBC 的多媒體內容，過去這些內容僅可見於執行微軟 Windows Media Player 的電腦上。如今在 8.10 的 Ubuntu 中，許多高品質的精選 BBC 內容將可透過內建的 Totem Movie Player 與 Rhythmbox 應用軟體，免費地提供給 Ubuntu 使用者觀賞。

在本次同時釋出的 Ubuntu 伺服器版本中，Canonical 加入了虛擬機器建構程式，根據該公司的說法，此功能可以在不到 5 分鐘內，在命令列下建立虛擬機器。此外， Ubuntu 伺服器新版提供完整的 Java 堆疊，包括 Apache Tomcat 6.0 與 OpenJDK，一舉使得 Ubuntu 在 Java 應用軟體開發與生產環境的部署上，成為更加強大的作業系統選項。

為了解決郵件伺服器安全性問題，ClamAV、SpamAssassin 已經可自主要套件庫下載。新版 Ubuntu 透過 DMRaid 也提供對 SATA 軟體 RAID 控制器的支援。在系統管理方面，Ubuntu 伺服器如今提供了私有目錄加密功能，並且可以在使用者登入時自動掛上 (mount) 檔案系統。

Canonical 營運長兼線上服務領導人 Jane Silber 表示，伺服器與桌面版本的新功能為桌面與組織用戶加入了更多可用性與彈性。來自 BBC 的內容是一大賣點，顯示 BBC 認可 Ubuntu 與 Linux 的主流接受度。

她認為，想要觀賞 BBC 內容的 Ubuntu 使用者只要用 Totem 播放軟體就可以觀看這些內容，無須使用瀏覽器造訪 BBC 網站。這種模式為服務供應商與內容打破了瀏覽器的框架。而伺服器版的完整 Java 堆疊支援，可說是因應了許多企業使用者的要求。

Silber 在聲明中指出，Ubuntu 8.10 伺服器版是一重要里程碑，彰顯出該公司雙軌釋出策略的價值。伺服器版納入雲端運算、虛擬化技術、郵件伺服器改良、Java 開發與部署，以及一系列可幫助系統管理者與開發者的服務。

Ubuntu 8.10 伺服器與桌面版本是 Canonical 6 個月釋出週期下的最新 Ubuntu 定期釋出，該釋出並伴隨著長達 18 個月的支援。儘管該釋出並未納入 Canonical 的長期支援 (Long Term Support)。目前僅 Ubuntu 8.04 提供長期支援。
