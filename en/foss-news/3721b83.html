開放源碼搭配廢棄電腦 解決電腦教學資源困境

Created at Friday, 08 October 2004 08:00      Last Updated on Friday, 09 March 2007 00:54 Written by 謝良奇
由市民、學生、專家、退休人員等人士自發組成的「夏威夷開放源碼教育協會（Hawaii Open Source Education Foundation）」，正在當地協助許多學校利用 Linux 等開放源碼軟體，重新付予廢棄電腦設備生命，同時解決學校因預算不足，學子無電腦可用的窘境。

Scott Belford 兩年前創設的夏威夷開放源碼教育協會，將推廣開放源碼軟體視為協會目標。該協會整理各方捐贈的廢棄電腦，灌上 Linux 等開放源碼軟體後，再送到夏威夷各地學校及非營利組織使用。捐贈而來的電腦，多半來自 Hickam 空軍基地。1992 年首次接觸 Linux 的 Belford，因故落腳夏威夷後，發現了利用開放源碼及 Linux 解決地區學校電腦教學資源、成本不足的方法。他認為：「我們有責任貢獻我們所獲得的事物。」他的目標是拉近技術專家與企業決策者間的距離。Belford 希望打破 Linux 支援不佳的壞印象。接受該協會 Linux 系統的單位，都有項共同職責，那就是維護系統，並參與開放源碼生態系統。事實上，他在決定發送 Linux 電腦前，都會先確認該地點是否能維護系統，以及對使用者提供教育訓練。這一點，是該協會成功的主要原因之一。該協會不僅提供成本低廉的設備，更發展出一套生態系統，以及接納開放源碼模式的文化。藉此，該協會創造出一套自發自為，且不斷擴展的模式，可供有意改採開放源碼的其它地區，做為借鏡。此外，該協會更在 McKinley成人社區學院等若干公立學校中設置 Linux 實驗室。夏威夷開放源碼教育協會和 Novell 已於九月份於檀香山舉辦為期一天的地區性發表會，宣傳開放源碼所帶來的益處。該發表會並邀請到開放源碼專家 Bruce Perens 發表演說。
