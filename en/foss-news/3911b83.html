Novell 與瑞士簽訂合約 3000 部公家系統改用 SUSE Linux

Created at Tuesday, 13 December 2005 08:00      Last Updated on Friday, 09 March 2007 00:45 Written by 謝良奇
Novell 宣佈與瑞士達成協議，將以該公司的 SUSE Linux 替換瑞士 公家機構的 3000 部系統。作業效率與成本是促使瑞士政府改用 Linux 的主因。

Novell 並未公開該合約的價格，只是表示這是瑞士政府的首宗正式 Linux 採購案。 瑞士聯邦政府資訊策略發言人（Delegate for Infor- mation Strategy ）表示，Linux 近年逐漸被各種政府採用，但這是首次聯邦層級的正式 Linux 採購案。該合約將擴及整個聯邦政府， 瑞士聯邦管理階層將普遍採納 Novell 的 SUSE Linux。 Roemer 表示，瑞士政府目前採用的是 IBM、Sun 和 HP 的 Unix 系 統與各種 Linux 散佈套件，希望加以標準化統一採用 SUSE Linux。 他表示 Red Hat 原先也在考慮之列，但最後還是 SUSE 勝出。 他說，瑞士政府在面對此採購案時，同時將私有與開放源碼軟體納入 考量，而決定採用開放源碼並沒有任何政治因素。Roemer 表示， Windows 是最為普遍的作業系統，現有的 Unix 系統最有可能加以更 換。 Novell 在 2 年前併購了德國的 SUSE Linux。此次瑞士的採購合約對於 Novell 意義重大。尤其是 Novell 在經歷過裁員、人事異動與 投資者和管理階層的緊張關係之後。 Novell Linux 與開放源碼行銷主管 Charlie Ungashick 表示，他們 經歷了痛苦的一年，從一家典型的軟體公司，轉型為 Linux 軟體公 司。但是他認為他們同時也有相當好的前景。他們的 Open Enterprise Server 在第一季已賺進 3100 萬的營收，並協助該公司取得市占率。
