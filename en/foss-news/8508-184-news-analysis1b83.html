184 期電子報－重要新聞回顧

Created at Tuesday, 08 November 2011 10:11      Last Updated on Tuesday, 08 November 2011 14:41 Written by 陳瑞霖／編述
自由軟體鑄造場電子報從 2003 年創刊時就一直關注自由軟體界發生的大事。並且依商業性、技術性、發展趨勢等原則，為讀者挑選數則重要自由軟體新聞。經由綜合各方說法及比較不同觀點，讓我們回顧前陣子發生的重要自由軟體新聞吧。

最近 UEFI Secure Boot 話題正紅，為了因應 Microsoft 可能運用 UEFI Secure Boot 功能排除競爭者，Linux 基金會也推出相關的報告。如果想看詳細的資訊，可以下載 PDF 檔的報告。

Mozilla 推出預載 Bing 搜尋引擎的 Firefox，如預期惹惱不少死忠 FLOSS 信徒。不過對於 Mozilla 來說，以雞蛋不放在同一個藍子的邏輯來看，推出 BIng 版 Firefox 是說得通的。早在 2010 年 Mozilla 與 Microsoft 就已經簽署財務協議，直到最近才弄出實際的客製化瀏覽器。正值 Mozilla 與 Google 換約前夕，Mozilla 的時間點很巧合。

Apple ALAC 格式開源，另外 Qt 專案變成由非營利組織 Qt project管理，成為真正的 Open Governmence。Linux 基金會針對消費性電子產品生產商推出長期支援的核心。

  1. 擔憂 UEFI Secure Boot 排除開放源碼軟體 Linux 基金會、Canonical、Red Hat釋出報告

  2. Mozilla 釋出以 Bing 為預設搜尋引擎的 Firefox 版本

  3. Apple 釋出無損音樂格式 ALAC Qt 邁向開放管理

  4. Linux 基金會宣佈針對消費者電子產品廠商推出長期支援的核心樹
