FSG 啟用開放源碼開發者網站 LDN

Created at Friday, 20 October 2006 08:00      Last Updated on Friday, 09 March 2007 01:42 Written by 謝良奇
旨在開發與推廣開放源碼軟體標準的非營利組織 FSG（The Free Standards Group，自由標準團體）推出了 LDN （Linux Standard Base Developer Network，Linux 標準基礎開發者網站），為 Linux 開發者提供建構與測試 LSB 相容應用軟體所需的完整資源，期望扮演開放源碼版 MSDN（Microsoft Developer Network，微軟開發者網站）的角色，並發揮同樣的作用。 LDN 是專為撰寫可移植之 Linux 應用軟體軟體開發者提供資訊來源的社群網絡，已經在 developer.freestandards.org 開始透過 beta 測試形式提供服務。FSG 的 LSB 則是一套希望在各種 Linux 作業系統與應用軟體間帶來相互可用性的規格標準，目前已受到各大 Linux 散佈套件商的支援。FSG 旗下成員包括 IBM、Intel、Novell、紅帽 (Red Hat)、紅旗 (Red Flag) 等。 FSG 並且與 O'Reilly Media 合作為 LDN 上的 Linux 應用軟體開發者提供服務，LDN 的用戶將可透過 Safari Books Online，取用到 O'Reilly 的客製 Linux 書庫。未來撰寫可移植、遵循 LSB 標準之 Linux 應用軟體的開發者，將無須再費心搜羅散置各處的相關資訊。 FSG 技術長 Ian Murdock 表示，微軟在微軟開發者網路上成果卓越，FSG 期望透過由下到上的社群開發模式，為 LSB 達成同樣的目標。使用 LDN 上的工具與資訊，開發者將能發展遵循 LSB 規格的軟體，並透過 FSG 的認證與測試服務確保其成果在各種 Linux 散佈套件上皆可運作無誤。Murdock 認為，為了讓 Linux 成為真正具有競爭力的平台，為開發者提供建構可移植之 Linux 應用軟體的中央資訊來源，是不可或缺的一步。 O'Reilly Media 創辦人兼執行長 Tim O'Reilly 認為，O'Reilly 與 FSG 都擁有共同的偉大目標，那就是透過鼓勵開放與創新，改造軟體世界的面貌。 HP 表示 LDN 是 Linux 邁向成熟應用軟體開發平台極為重要的成果。IBM Linux 技術中心主任 Kathy Bennett 則認為，有意撰寫符合 LSB 規格之可移植軟體的開發者如今可以自由取用獨立於各廠商之外的中立資源，這一點為推動 Linux 成為更具競爭力的運算平台，踏出了新的一步。同時，LDN 正好回應了日前 Linux Watch eweek 主編 Steven Vaughan-Nichols 疾呼 Linux 社群應該學習 MSDN 精神，並加強平台與服務以提供開發者支援。 LDN 提供連接書籤讓用戶匯入 del.icio.us 現有書籤並分享給網絡上的其它用戶，也有建構可移植之 Linux 應用軟體的資源目錄，其中包括 LSB 應用軟體開發教學與其它資訊如來自 Linux Magazine 的內容、軟體開發套件、LSB 指導工具、驗證服務，更重要的是 LSB 工作小組與社群的支援。 相關網址：
