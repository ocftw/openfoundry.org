Novell、IBM 聯手出擊 共推 SUSE Linux 企業版整合套件

Created at Thursday, 19 October 2006 08:00      Last Updated on Friday, 09 March 2007 00:43 Written by 謝良奇
Novell 日前推出的 SUSE Linux 企業版整合套件 (Integrated Stack for SUSE Linux Enterprise) 包含了 Novell 的 SUSE Linux 企業版伺服器以及特別針對 IBM 系統最佳化的一系列開放與私有源碼軟體，是專為中小型企業設計的軟體套件，可提供一次到位的檔案、列印、Web 以及資料庫系統，為目前正在使用微軟 Windows 系統的組織，帶來更大彈性並可加強其 IT 架構安全性、可靠性與執行效能。

這套包含 SUSE Linux 企業版 10 的軟體套件可在搭配 AMD Opteron、Intel Pentium IV 與 Xeon-based 的 IBM System x 或 BladeCenter 伺服器上執行。 SUSE Linux 企業版整合套件內含企業級私有與開放源碼軟體元件，其中有採用了 Apache Geronimo 技術的 J2EE 應用伺服器─IBM WebSphere 應用伺服器社群版、支援關聯式與純 XML 的混合型資料庫系統 IBM DB2 Express-C、管理防火牆、SSH 連線、用戶與群組等功的 Centeris Likewise 管理套件 2.0。 Novell 開放平台方案副總裁兼總經理 Roger Levy 表示，隨著這套整合套件的推出，用戶可以快速地部署出資料庫與客製化應用軟體的平台。由於 SUSE Linux 企業版與生具來的相互整合能力，用戶很簡單快速地就可以把這套軟體套件與企業現有架構相互整合，大大提高彈性並降低成本。 根據 IBM 的資料，針對 100 名用戶與一部雙 CPU 系統，這套解決方案的授權費加上一年期的支援共 13867 美元。微軟類似的系統則要價 20990 美元。此外，假如企業自行負責支援，更僅需 349 美元。 相較於紅帽 (Red Hat) 的應用軟體套件 (Application Stack)，Novell 與 IBM 合作推出的此一套件不僅在價格上仍具有優勢，SUSE Linux 企業版整合套件採納開放標準，並且融合了兩大軟體商的私有與開放源碼軟體，在功能性以及與 Windows 整合能力上都有更為突出的表現。 Centric CRM 執行長 David Richards 表示，SUSE Linux 企業版整合套件為他們的開放源碼 CRM 用戶帶來理想的應用軟體平台。這套軟體特別針對中型企業用戶考量，隨著企業成長其功能也可輕鬆擴充。這樣的表現全都有賴於該套件整合了從硬體、作業系統到中介軟體等各階層的最佳元件。
