據傳微軟 Office Web 應用軟體 可相容 Mac OS X、Linux 與 iPhone

Created at Thursday, 20 November 2008 08:00      Written by 謝良奇/編譯
有消息傳出，微軟即將推出的 Office Web，也就是輕量級的 Office 線上服務版本，將可供 Mac OS X 與 Linux，甚至是 Apple 的 iPhone 用戶使用。對於非 Windows 平台的用戶而言，這顯然是個好消息，但是對照於之前完全相反的說法，加上 Office Web 開放給其他平台用戶的實際做法仍舊不明，這些訊息仍有待微軟釐清。

在微軟主導的 Channel 10 網誌上，名為 Sarah Perez 的人日前發表了 Office Web 的系統需求。根據 Perez 的文章，此一 Word、Excel、PowerPoint 與 OneNote 線上版本，不僅可在 Windows 上透過微軟的 Internet Explorer (IE) 存取，更可在 Mac OS X 與 Linux 上，使用 Mozilla 的 Firefox 以及 Apple 的 Safari。

Perez 表示，由於 Apple 的 iPhone 包含了瘦身過的 Safari 版本，因此 iPhone 用戶也能夠使用 Office Web。

Firefox 有 Mac OS X、Linux 與 Windows 的版本，Safari 則有 Mac OS X 與 Windows 版。根據網路計量公司 Net Applications 本月稍早的報告，Firefox 與 Safari 共佔 10 月份瀏覽器使用的 26%。微軟 IE 則佔了 71%。

這個說法與前些日子參加於洛杉磯 (Los Angele) 所舉辦 PDC 2008 的媒體所收到的消息，可以說完全相左。當時與會者首次得窺此一 Office 14 的 Web 應用軟體。Office Web 應用軟體在會場中透過 Internet Explorer 8 與 Mozilla Firefox 3，在基於 Windows 7 的虛擬機器上展示。展示中，Open XML 格式文件與試算表被儲存在 Office Live Workspace 上。透過 Office Live Workspace 這套服務，Windows 的 Office 應用軟體得以將文件儲存在微軟的雲端伺服器。

雖然理論上 Live Workspace 的頁面可以在不同作業系統上的瀏覽器呈現，不過此一服務目前並未附加於 Mac 最新的 Office 2008 套裝軟體上。Office Web 微軟目前使用的用戶端執行時期架構 Silverlight 並不支援存取本地電腦檔案系統，儘管 Mac OS X 上有 Silverlight，也不能夠作為 Office Live Workspace 的儲存替代方案。除非這些網路應用軟體使用完整的 .NET Framework 執行時期，然而，此一方案僅有 Windows 版本。

有趣的是，針對此一網路應用軟體可否在 Linux 上執行的問題， Channel 10 上的 FAQ 回覆指出，不管使用的是何種作業系統，只要有支援的網路瀏覽器即可。而且並不需要 Silverlight。但是在 PDC 上聽到的說法是，Word 與 Excel 網路應用軟體都需要 Silverlight，以便確認用戶系統所安裝的字型。

由於文章發佈者 Sarah Perez 並非微軟官方代表，這些訊息的可靠度有多少仍未獲得微軟方面的回應。此一線上軟體與微軟下一個升級版本 Office 14 同時公開，儘管 Office 14 預定於 2009 年稍晚推出，微軟並未提供 Office Web 的推出日期。此外，是否收費與收費方案也未公佈。

面對微軟這一次可能跨足 Linux 與 Mac OS X 平台，有觀察家指出紅帽 (Red Hat) 會是受到威脅的主要廠商。觀察家指出重點並不在紅帽的 Linux 桌面業務。一旦微軟成功跨出封閉的生態系統，將其技術開放到 Linux 與 Firefox 等替代平台上，微軟將其吸引力擴大至 Windows 外，最終將打亂紅帽的 Unix 與 Weblogic 替代商機。

早在微軟與 Novell 簽訂專利與相互操作性協議，這樣的威脅便已經展開。微軟將技術擴展到更為廣大的技術堆疊後，會成為紅帽的更大威脅。因為此舉破壞了紅帽的主要賣點，選擇。藉由讓潛在用戶先行試用微軟的技術，微軟收納了 Linux，讓 SharePoint、Office 等軟體在 Linux 上執行，說服用戶轉向 Windows 上提供的加強版應用軟體，也就更為輕鬆。

對於紅帽而言，最好的因應策略可能是儘早行動，切入微軟的版圖，而非等到 Unix 市場完全煙消雲散，或 JBoss 擊敗 Weblogic 與 Websphere。如果這樣，紅帽等到的將是在大門外敲門的微軟。
