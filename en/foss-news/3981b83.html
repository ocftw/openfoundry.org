OpenBSD 4.0 釋出 添加新硬體支援

Created at Thursday, 19 October 2006 08:00      Last Updated on Friday, 09 March 2007 00:38 Written by 謝良奇
OpenBSD 團隊領導人 Theo de Raadt 不久前宣告了 OpenBSD 4.0 的釋出消息。以安全防護完善為特色的開放源碼作業系統 OpenBSD，在 4.0 版中新增了各種硬體支援，並且緊密地整合了 X.Org。

OpenBSD 是採用 BSD 授權的開放源碼作業系統，在所有的 BSD 用戶中，使用 OpenBSD 約佔 30%，是使用者第二多的 BSD 作業系統。OpenBSD 一向強調高規格的安全防護，內建的加密與防火牆支援受到電腦安全社群的青睞。 Theo de Raadt 表示，他們榮幸地宣佈 OpenBSD 4.0 的正式釋出。這已經是 OpenBSD 第 20 次的 CD-Rom 釋出。對於 OpenBSD 在 10 年內僅有一次預設安裝存有遠端漏洞的記錄，他們感到相當自豪。Theo de Raadt 除了身為 OpenBSD 團隊領導人外，也因為其對於自由軟體的強烈態度聞名，他更是自由軟體基金會 (Free Software Foundation, FSF) 2004 年自由軟體成就獎的得主。 OpenBSD 4.0 針對一系列網路硬體設備提供新驅動軟體，改善了內部 IPSec 的功能，包括加入 IPv6 支援，並支援 Intel SpeedStep 與 AMD PowerNow 的處理器頻率延展技術。此外，本次釋出對各種使用 Redboot 啟動載入器的 ARM 設備，以及採用 UltraSPARC III 的設備與 Zaurus SL-C3200 提供了硬體支援。 除了硬體之外，OpenBSD 4.0 的重大特點之一是正式引入 OpenRCS。OpenRCS 是以 BSD 授權的 GNU RCS 相容重新實作，以便改善安全性與可移植性等。 OpenBSD 4.0 光碟能夠在 i386、amd64、macppc、sparc 與 sparc64 平台上開機，其它平台則需從軟碟、網路等途徑開機。 OpenBSD 4.0 緊密地整合了 X.Org 6.9.0，雖然並未納入 Ubuntu 與 Fedora 近來採用的 X.Org 7，但 De Raadt 表示，用戶安裝時可輕鬆安裝 X.Org，更希望用戶能嘗試他們為 OpenBSD 客製化後的 xdm。他進一步表示未來會切換到新的 X.org 7 上。 各種平台的 OpenBSD 4.0 已可線上訂購或從 FTP 自由下載。一如往常，OpenBSD 每次釋出都會伴隨著主題圖案與歌曲，藉此傳達 OpenBSD 開發團隊的想法，這次的釋出也帶來了一連串的宣傳活動，像是 OpenBSD 4.0 官方主題曲 Humppa Negala，以及由 OpenBSD 漫畫角色 Pufferix、Bobilix 領銜主演的漫畫作品。
