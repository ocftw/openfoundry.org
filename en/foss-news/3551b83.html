模組化 Wordpress 平台 提供多樣化 plugins

Created at Tuesday, 21 June 2005 08:00      Last Updated on Friday, 09 March 2007 01:03 Written by 謝良奇
Wordpress 已經是 Blogger 架設自家 Blog 的主要 Blog 平台之一， Wordpress 模組化的設計，可以輕易地加入 plugins 加強原有功能。 plugins 不僅安裝簡便，自行撰寫客製化的程式碼也十分簡單。

Wordpress 是先進的語意化個人出版平台，以 PHP、MySQL 撰寫，採 用 GPL 授權，強調使用者體驗與網路標準為特色。Wordpress plugins 為可強化原有 Wordpress 平台的程式碼，由於 Wordpress 的模組化 設計，經適當撰寫的 plugins 可在無須更動 Wordpress 核心程式的 情況下，置入 Wordpress 平台中。 Wordpress plugins 功能種類多樣，可分為管理、設計與外觀、圖像聲訊、佈告（Post）、連接與其他等類。管理類的 plugins 能客制化管理工具、垃圾信防治、評論、統計、語法標示、同步供稿 （Syndication）。連接類的 plugins 會更改連接的預設行為。其他 類的 plugins 則提供如顯示天氣等外部網站資訊的各項功能。 一般而言，安裝 plugins 的過程只需把 plugins 置於 wp-content/ plugins 路徑下，登入 Wordpress 並在主選單 Plugins 連接下選取 所需之 plugins，並加以啟動（Activate）即可。 Wordpress 領導開發者 Matthew Mullenweg 日前開發出新 Wordpress 搜尋網站 search.wordpress.org。根據 Mullenweg 表示，過去採用 資料庫自行撰寫搜尋功能或採用 Google網站搜尋（site-search）的 結果都不甚完善，最近透過 Yahoo! 開發者網路（Yahoo! Developer Network）提供的 API，讓整個開發過程簡化許多。Mullenweg 描述整個開發過程是異常的簡單，撰寫過程只花去他數十分鐘。
