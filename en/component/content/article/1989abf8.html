Sun 釋出整合式開放源碼平台 GlassFish Portfolio

Created at Tuesday, 17 February 2009 08:00      Written by 謝良奇/編譯

Sun 宣佈釋出基於 GlassFish Java 應用軟體伺服器的開放源碼 Web 應用軟體平台 Glassfish Portfolio， 提供 JavaEE 應用軟體伺服器、AMP (Apache、MySQL 與 PHP) 堆疊以及入口軟體和管理工具。此一整合平台可作為企業更具吸引力的替代方案。如今開放源碼應用伴隨著整合不同元件的時間與成本，以及支援。根據 Sun 表示，較之私有方案，GlassFish 平台可節省 90% 的價格效能比。

GlassFish Portfolio 強調產品化並預先安裝的開放源碼技術，有助於降低企業採納開放源碼的門檻。開放源碼的隱藏成本之一正是實作與支援開放源碼所需的技能、時間與資源。再者，企業級的開放源碼平台囊括一整套技術堆疊，整合與設定組態將構成另一個成本。各個開發團隊可能以不同方式設定堆疊組態，於是增加企業維護與支援的整體成本。

Sun 中介軟體行銷副總裁 Mark Herring 表示，該公司認為此一產品重要之處在於，現今客戶建構應用軟體時，若打算採取開放源碼途徑，面臨到的將是一場整合惡夢。而 Sun 只是為客戶完成此一工作。

GlassFish Portfolio 平台提供標準化的整合堆疊，有助簡化管理與降低成本。透過整合的 AMP 堆疊與 Java，Sun 的 Glassfish Portfolio 有助該公司擴展其核心 Java 應用軟體伺服器，提供企業應用軟體執行的中介平台。此一釋出可視為對於紅帽 (Red Hat) JBoss 與 IBM Websphere 等中介軟體競爭對手的回應。

GlassFish 平台包含了完整的 LAMP (Linux、Apache、MySQL、Perl/PHP/Python) 堆疊，該堆疊除提供輕量級 Web 解決方案，更包括 Tomcat、記憶體物件快取系統 Memcached，並支援 PHP、Ruby 和 Java。基於 Liferay 的 GlassFish Web Space Server，提供了網站開發功能。GlassFish ESB 則是一套輕量級的企業服務匯流排，可供企業部門與 SOA 部署之用。

儘管並不預期用戶立刻從已有的 WebSphere 與 WebLogic 平台轉移過來，Sun 認為組織在目前的經濟情勢下，將會重新看待開放源碼在價格效能上的優勢。

GlassFish Portfolio 中除了 Sun Enterprise Manager 外，都是開放源碼元件。Enterprise Manager 是管理 GlassFish Portfolio 的封閉源碼產品。Herring 表示，該公司有意將所有軟體開放源碼，不過可能也有某些額外功能不在其中。

Herring 指出，從該公司的觀點，組織中有資料，也有 Web 瀏覽器的呈現層，另外則有存取資料加以用有意義的方式呈現的軟體，而 Glassfish Portfolio 的目標即成為此一中間層。對於 Sun 而言，這是相當關鍵的地位，畢竟這是實際提供應用軟體的層次。

除了供應應用軟體的能力外，Glassfish Portfolio 同時也提供入口應用軟體，也就是 GlassFish Web Space Server。Herring 表示，Web Space Server 有 90% 以 Sun 所貢獻的開放源碼 Liferay Portal 做基礎。

GlassFish Portfolio 以訂閱定價方式銷售，價格為每部伺服器 999 美金起。除了 GlassFish Portfolio 外，Sun 也提供與 Ericsson 共同開發的 GlassFish Communications Server，此為供匯合式多媒體應用軟體使用的電信級基礎。服務供應者可為企業與用戶設備，提供如 VoIP、即時傳訊與互動式遊戲。
