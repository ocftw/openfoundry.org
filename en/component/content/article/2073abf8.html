Mandriva Linux 2009 Spring 釋出

Created at Tuesday, 05 May 2009 08:00      Written by 謝良奇/編譯

Mandriva 釋出了 Mandriva Linux Spring 2009 的最終版本，提供 KDE 4.2.2 作為預設桌面，開機速度加快 25%，支援更多 netbook，並提供更強大的網路與安全工具。

Mandriva Linux 2009 Spring 提供了三種不同版本：Free、One 與商業的 Powerpack 版本。Free 版本內僅包含開放源碼軟體，One 是附加私有驅動程式及 plugins 的 Live CD 版，商業版的 Powerpack 分為盒裝與簡裝版，要價分別為 89 與 69 美金。商業版 Powerpack 提供支援服務以及額外的應用軟體，根據 Mandriva，其中包括額外的音訊與視訊 codecs、VMware 虛擬軟體、Google 應用軟體。

Mandriva Linux Spring 2009 所使用的核心版本，已升級到  2.6.29，並且支援超過 70 種語言。除了在開機時間上，加快了 25%，Mandriva 2009 也提供回應性更高的的關機、待命、休眠與回復作業。

Mandriva 在去年 10 月釋出了 Mandriva Linux 2009 的第一個 Fall 版本，主要以預設桌面 KDE 4 為號召，這一次在 Spring 版本中，預設的 KDE 已經升級到 4.2.2，也提供了 Phonon 多媒體框架、Plasma 桌面框架、Plasmoid widgets 等多項加強功能。對於其他桌面環境，Mandriva 則支援了 GNOME 2.26、X.Org Server 1.6，以及輕量級的 LXDE 與 XFCE 4.6 桌面。

正如大部分消費者 Linux 散佈套件專案，Mandriva 在此一最新釋出中，也開始關注逐漸成長的 netbook 用戶群。Mandriva Linux 將之前對 Asus Eee PC 的支援加以擴展，如今支援所有現有的 Eee PC 型號，並支援 Acer Aspire One、MSI Wind 等眾多普遍的 netbook。

以 netbook 為導向的 LXDE，是預設的輕量級桌面，此外還有可供替代的 XFCE 4.6。LXDE  提供了傳統的啟動選單 (launcher menu) 和功能面板 (panel)、PCMan 檔案管理、工作管理程式、顯示設定工具、影像瀏覽程式、記事應用軟體、檔案庫管理應用軟體。

Mandriva Linux Spring 2009 內含升級後的應用軟體包括 Firefox 3、OpenOffice 3.0.1、VirtualBox 2.2.0、Wine 1.1.19 與最新的 Phoronix Test Suite 和 Songbird 音樂播放程式。其中特別的是，跨平台整合開發環境 Qt Creator，該環境支援來自 Nokia 的 Qt Software 的 Qt. 4.5 開發框架。

Mandriva Linux 2009 Spring 的網路中心支援了進階的網路設定，以及為多個網際網路供應商預先設定的網路工具，整合了新的網路設備與不同無線使用規範 (Regulatory domain) 支援。另外，Msec 安全框架經過重新設計，加入 plugins 支援以及新的 GUI。其他 Msec 相關改善包括了：自定安全等級支援、桌面問題通知、進階日誌等。

伴隨最新 Linux 核心，該散佈套件如今支援 Ext4 檔案系統，同時升級 Syslinux 為 3.75 版，其中提供了新的 Hardware Detection Tool (HDT) 模組，可在系統開機前偵測所有硬體元件。
