主攻新手的 Linux 套件：Xandros Desktop 和 Lycoris Desktop_LX

Created at Thursday, 23 September 2004 08:00      Last Updated on Friday, 09 March 2007 00:21 Written by 謝良奇
Xandros Desktop 和 Lycoris Desktop/LX 是兩套主打 Linux 新手和桌面市場的 Linux 套件。Xandros Desktop、Lycoris Desktop/LX 剛好都以 KDE 桌面環境為基礎。 早期稱為 Corel Linux 的 Xandros Desktop，可算是有點歷史的散佈套件。Xandros Desktop OS version 2.5 的網頁介紹描述，只要四次點選，便可自動磁碟分割並安裝。不過《PC World》雜誌的 Matthew Newton實際測試時，在磁碟分割上卻遇上麻煩。主要是 Xandros Desktop 無法縮減 NTFS 分割區，以及不完善的系統訊息。

Xandros Desktop 第一次啟動會有程式引導設定印表機及軟體更新。Xandros 基於 KDE 3.1.4 版，KDE 內建的檔案管理器及瀏覽器 Konqueror，則被 Xandros Desktop 以Xandros File Manager 取代。Newton 認為這是 Linux 上最佳的 Windows Explorer 相似軟體。套件內含 Mozilla、OpenOffice.org、Kopete，以及若干處理數位相機與數位音樂的軟體。

Xandros Desktop 可用的軟體相當豐富，內建的 Xandros Networks工具，可以下載軟體更新，該工具可以取用 Debian 專案的各項線上套件。

Xandros Desktop 還提供 PowerTerm 版本，該版本整合 Ericom 的PowerTerm InterConnect for Linux，該系統用戶可從 Linux 環境連接大型主機上的舊有企業系統。

Lycoris Desktop/LX 用的是 KDE 3.2.3 版。Newton 測試後表示該散佈套件的安裝程式過時，步驟過於繁瑣。Desktop/LX 提供 Lycoris Web Suite─也就是 Mozilla 1.7─用以處理電子郵件及網路瀏覽。Desktop/LX 選用 KOffice 而非 OpenOffice.org，對於熟悉微軟 Word 的用戶，KOffice 的 KWord 不像 OpenOffice.org 的 Writer 那麼好上手。
