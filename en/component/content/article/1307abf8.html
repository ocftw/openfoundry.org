Everex 低價 Linux 個人電腦在 Wal-Mart 銷售一空

Created at Thursday, 15 November 2007 08:00      Last Updated on Monday, 26 November 2007 02:27 Written by 謝良奇/編譯

萬聖節前夕 Wal-Mart 開始銷售 Everex 的低價 Linux 個人電腦 TC2502 Green gPC，售價為 199 美元。Wal-Mart 日前表示這批 Linux 電腦已經銷售一空，該零售商預計補貨後會重新上架。

Wal-Mart 本月初開始在線上銷售名為 Everex gPC 的 Linux 個人電腦，當初該公司一開始進了約 1 萬台 gPC，兩個星期後已全數售出。Wal-Mart 發言人不願透露確切的銷售數字，只表示 gPC 是 Wal-Mart.com 過去數週內，銷售表現最好的桌面電腦之一。該公司並未對何時該產品會重新上架，提供立即答覆。

gPC 預先安裝了一套 Linux 作業系統 gOS，配備 Via Technologies 的 1.5 GHz 處理器，以及 80-Gbyte 硬碟、512 Mbytes 的 DDR 2 記憶體，與揚聲器和 DVD-ROM/CD-RW 光碟機。gPC 不包含螢幕。

軟體方面，gPC 安裝了免費的生產力軟體套件 OpenOffice.org 2.2，該套件包含文字處理、試算表與簡報軟體，日前 IBM 以實際行動表達了對該套件的支持態度，發行了以 OpenOffice.org 為基礎的免費辦公室軟體 Lotus Symphony。

Everex 是一家正企圖搶占低價位系統供應商地位的台灣電腦廠商。Wal-Mart 上販售的 Everex 另一款個人電腦 Everex Impact GC3502，執行微軟 Windows Vista 基本版，售價為 298 美元，但銷售情況不如 gPC。

Everex 行銷總監 Paul Kim 表示，他們本來針對黑色星期五（Black Friday。感恩節過後的周五，每年的感恩節到聖誕節之間是美國折扣季）後的銷售檔期保留了一批 gPC，不過看到如此的銷售盛況，他們決定要提前釋出。同時，Everex 正在製造更多的低價 Linux 桌面電腦。

除了 Wal-Mart 之外，Everex 將擴展該產品的銷售管道。Kim 表示，該產品已經開始由線上 Linux 系統零售商 ZaReason 販售，該零售商更提供 Wi-Fi 與 2GB 記憶體擴充的系統升級選項。線上電腦與技術零售商 NewEgg，根據 Kim 的說法，不久也將推出該產品。

一位消費者在 Wal-Mart 討論區上寫道，一部可以上網、發送電子郵件、即時通訊與進行一般文字處理工作的低價電腦，這個產品是為有這種需求的消費者量身打造的。在這個價格帶，找不到相匹敵的產品。

另一個 gPC 受到歡迎的原因，是上面執行的 gOS Linux 作業系統。基於易用的 Ubuntu 7.10 Linux，該系統採用了 Enlightenment E17 桌面環境，加上預先設定的 Google 應用軟體，提供了可能是最容易使用的 Linux 桌面體驗。
