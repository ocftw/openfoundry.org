Sun 與 Canonical 合作 Ubuntu 將搭載 Java EE

Created at Saturday, 11 November 2006 08:00      Last Updated on Thursday, 08 March 2007 08:17 Written by 謝良奇
Sun 與 Ubuntu Linux 贊助商 Canonical 日前共同開發合作宣言， Ubuntu Linux 散佈套件將內含 Sun 的應用伺服器 Java Enterprise Edition 5。而 Ubuntu 也已經在 Sun 的 Opteron 與 Niagara 伺服 器上取得認證，企業將可自 Canonical 取得在 Sun 的硬體上執行 Ubuntu Linux 的支援。 Sun 與 Canonical 表示，隨 Ubuntu 一同散佈的首套 Java 應用伺 服器 Java EE 5 將讓開發者更快速建構先進的企業 Java 應用軟體。

Ubuntu 創始人 Mark Shuttleworth 表示，Ubuntu 與代號 Glassfish 的 Java EE 5 之間的結合讓開發者得以使用最新的開放源碼框架來 開發下一代的 Web 2.0 應用軟體。

在此同時，Canonical 也對企業市場展現更積極的企圖心，該公司表 示 Ubuntu 將取得 Sun 的 x64 硬體的認證，並提供支援。企業未來 將可在 Sun x64 系統上，執行 Ubuntu，並取得來自 Canonical 的 支援。

除了 5 月時 Sun 與 Canonical 已經宣佈 Ubuntu 移植至 Niagara Sparc T1 處理器，Ubuntu Long Term Support 6.06 版也釋出於 Sun 的 T1000 及 T2000 伺服器。Sun 與 Canonical 正致力讓 Ubuntu 也能通過 Sun 基於 Opteron 處理器的 Galaxy 伺服器的認證。

目前 Ubuntu LTS 6.06 已經通過 X4100、X4200 伺服器與 Ultra 20、 Ultra 40 的認證，預料兩家公司將進一步讓 Ubuntu 通過全系列 Galaxy 機器的認證，包括 X2100、X2200 與 16 核心的 X4600 伺服 器。

Sun 企業 Java 平台副總裁 Karen Tegan Padir 表示，Sun 與 Ubuntu 社群合作，正是 Sun 對開放源碼一貫支持決心的展現。透過在 Ubuntu 散佈套件上納入 GlassFish Java EE 5 應用伺服器，將可讓開發者 更輕易取得最新企業 Java 平台。加上 Sun Fire x64 產品的認證與 支援，Ubuntu 將成為吸引企業用戶的選擇。
