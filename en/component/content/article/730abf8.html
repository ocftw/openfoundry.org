Aras 主打微軟企業開放源碼牌引爭議

Created at Friday, 26 January 2007 16:00      Last Updated on Thursday, 05 July 2007 23:15 Written by 謝良奇
產品生命周期管理軟體供應商 Aras，將旗下的 Innovator 軟體採取開放源碼模式釋出，並稱之為所謂的微軟企業開放源碼 (Microsoft Enterprise Open Source)。 這套產品生命周期管理 (PLM) 軟體，能控管新產品從概念發想到上市之間的種種程序。Windows 平台的開放源碼軟體並不少見，然而這一類型的開放源碼生產系統多半會採用 Linux。此外 Innovator 採用的授權模式 Ms-CL，加上僅僅支援 Windows，讓外界不禁對該公司開放源碼的決心產生不少疑問。

位於麻薩諸塞州的 Aras 以開發軟體開發管理軟體與自動化汽車、半導體製造的設計異動程序管理工具著稱，其 Innovator 軟體包含 3 大部份：計劃管理，供追蹤軟體開發專案進度之用；產品工程，提供異動管理功能，協助設計者確定開發目標；品質規劃，協助企業管理風險。除了獨立使用外， Innovator 可以搭配 Windows、Unix 或基於 i5/OS 的現存 ERP 解決方案一同使用。該軟體也提供 XML/SOAP 網路服務可供輸出資料。

Innovator 執行平台為 Windows Server 2003 並需要 NET 2.0 與 SQL Server 2005。該軟體以微軟的 .NET 為基礎，並且可與微軟其它產品如 Office、Office SharePoint 伺服器 2007 等結合。

Aras 以微軟的共享使用授權─微軟社群授權 (Microsoft Community License，Ms-CL) 釋出 Innovator 8，如今使用者可以從 Aras 的網站，或微軟的開放源碼專案網站 Codeplex，免費下載 Innovator 8。

Aras 行銷副總裁 Marc Lind 表示，依照 Aras 的新策略，組織可以善用現有基礎建設與技術能力，免去轉移到其它平台耗費的成本、時間與人力。Lind 認為，由於該軟體的取得成本是零，大大降低了企業端的風險。不過 Aras 不提供支援給免費下載的用戶，希望獲得支 援的用戶必須購買會員套件，企業級套件為每月 8300 美元，小型企業則為每月 4100 美元。

包括 Ms-CL 在內的各微軟共享使用授權如微軟許可授權 (Microsoft Permissive License，Ms-PL)、微軟委任授權 (Microsoft Reference License，Ms-RL)，都不曾通過開放源碼協會 (Open Source Initiative，OSI)的認證。由於 Innovator 8 僅以 Ms-CL 授權釋出，加上只能在 Windows 平台使用，對於 Aras 採取開放源碼此事著實大打了折扣，開放源碼界是否真視 Innovator 8 為開放源碼軟體，相信仍有待商確。

對此 Aras 總裁 Peter Schroer 顯得相當自信，他表示將 Innovator 解決方案以企業開放源碼在 Windows 平台釋出，他們提供簡單的軟體授權，並且讓 Innovator 得以與企業現有技術接軌。

Aras 的做法看來也得到微軟的肯定。微軟的平台技術策略總經理 Bill Hilf 就認為，他們很支持 Aras 在微軟平台上提供企業開放源碼解決方案的這一步，可以預期當更多產業夥伴以不同授權模式運用微軟平台價值後，微軟合作夥伴生態系統將有驚人成長。
