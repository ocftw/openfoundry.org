Ubuntu Linux 宣佈與 Intel 合作 計劃推出行動裝置版本

Created at Wednesday, 16 May 2007 15:15      Last Updated on Tuesday, 03 July 2007 23:26 Written by 謝良奇
Ubuntu Linux 即將出現你身旁的手機裡。Ubuntu 技術長 Matt Zimmerman 宣佈計劃打造 Ubuntu Linux 散佈套件的行動裝置版本，而合作對象將是 Intel。Ubuntu 開發者大會上，開發人員將討論新 Ubuntu 行動與嵌入式專案 (Ubuntu Mobile and Embedded project) 的細節。

Ubuntu 開發人員將與 Intel 合作，開發出 Ubuntu Linux 的行動裝置版本。根據 Ubuntu 網站的消息，Intel 正在計劃代號 Silverthorn 的新低功率處理器，以及晶片組架構，讓行動裝置也能擁有完整的網際網路功能。

距離 Ubuntu 發佈這項消息約 1 個月前，GNOME 才剛在嵌入式 Linux 大會 (Embedded Linux Conference) 上，宣佈了一項與 Ubuntu 相似的行動平台計劃：GNOME 行動與嵌入式協會 (GNOME Mobile and Embedded Initiative) 。該項計劃邀請有興趣參與開發基於 GNOME開放源碼元件的行動裝置應用軟體的開發者與製造商加入。Intel 也是 GNOME 這項計劃的支持者之一。和 GNOME 的計劃相似，Ubuntu 計劃讓開發者與 Intel 等硬體製造商共同合作。

不久前 Dell 宣佈計劃預先在其 Linux 桌面與筆記型電腦上安裝 Ubuntu Linux。Dell 表示這項決定是為了回應消費者對於 Ubuntu Linux 表達的高度興趣。隨著 Dell 的計劃，加上這次踏入行動運算領域，Ubuntu Linux 看起來將成為 Linux 社群第一個主流成功典範。

Zimmerman 表示，將改變人們通訊與合作方式的小型網際網路手持裝置的崛起，是 Ubuntu 此計劃的主要動力。這些裝置為開放源碼軟體投入新的需求，同時需要創新的圖形介面、更好的電力管理與互動能力。

為了因應這些技術挑戰，Zimmerman 表示 Ubuntu 將與 Intel 合作，為這項專案在技術、人員與專業上，帶來重大的貢獻。

Ubuntu 行動裝置版本預定會在 10 月，連同新的 Ubuntu 7.10 版一起釋出。

Canalys.com 分析師 Nick Spencer 認為，雖然 Linux 在智慧型手機上的運用比率相對仍低，僅佔 2006 年智慧型手機作業市場約 6%，隨著越來越多手機製造商從私有手機作業系統出走，對於開放源碼的需求將會上升。Spencer 並且預測這些需求主要來自亞洲，特別是中國。
