Google 釋出 Google Picasa 3 的 Linux beta 版本

Created at Thursday, 09 October 2008 08:00      Written by 謝良奇/編譯

Google Picasa 3 的 beta 版本如今已出現在 Linux 平台上。Linux 用戶可以使用新的 Picasa 3 軟體來編輯、歸類與上傳影像。

Picasa 3 beta 版已經釋出於 Windows 平台多時，Linux 版本終於追趕上 Windows 的腳步。獨缺 Mac OS X 版本，可能讓 Mac 用戶感到失望。Google 的 Picasa 產品經理 Mike Horowitz 則表示，他們很重視 Macs，他們一直設法讓用戶有更好的使用經驗，他們正在考慮中。

Google 程式設計者 Lei Zhang 在網誌中寫到，新釋出的版本加入了潤飾 (retouching) 工具，自動同步個人電腦與 Google Picasa 網站所存放照片的功能，以及拼貼 (collage) 模式，可用以將多張照片組合為海報大小的照片集。而且，他說，新版本更快了。

儘管如此，新版仍缺了 Windows 版本有的電影製作 (movie maker) 功能。這個功能可以將照片轉為搭配音樂的幻燈片展示 (slideshow)。製作出來的幻燈片更可以上傳到 YouTube 等網站。

新釋出的版本使用了 Wine 以及開放源碼軟體層，可將該程式的 Windows 指令，轉為 Linux 的指令。今年迄今，根據 Google  表示，該公司已貢獻 Wine 專案約 850 個程式修補。由於 Wine 仍致力於提供更好的影片支援功能，或許這是 Picasa 3 beta Linux 版本缺少電影製作功能的可能原因。
