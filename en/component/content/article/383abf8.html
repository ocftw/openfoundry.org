MySQL Enterprise 提供監測服務 鎖定企業用戶

Created at Thursday, 19 October 2006 08:00      Last Updated on Friday, 09 March 2007 00:49 Written by 謝良奇
MySQL AB 日前於倫敦舉辦的年度北歐使用者大會上，宣佈推出旗艦級商業訂閱服務 MySQL Enterprise，反映出這家全球最普及開放源碼資料庫廠商，對於企業用戶市場的積極策略，MySQL Enterprise 包含了新的自動化主動監看技術，能協助企業用戶監測與調整線上的 MySQL 資料庫系統。Gartner 的首席分析師 Colleen Graham 儘管肯定 MySQL 等開放源碼資料庫廠商的迅速成長，但要追上 IBM、Oracle 等傳統資料庫大廠仍有相當長的距離。

MySQL AB 執行長 Marten Mickos 表示，管理 MySQL 資料庫應該跟開發 MySQL資料庫同樣簡單，MySQL Enterprise 提供企業管理與維護高效能、可靠性高的應用軟體所需的一切。新的監看與諮詢服務能幫助從 Web 2.0 新興公司到企業 IT 部門內的 MySQL 資料庫管理者。根據 Forrester Research 的報告，自動資料庫管理能降低複雜度、減少人為錯誤、改善整體生產力並減低成本。企業透過減少管理負擔，將可節省 15% 以上的成本。 新的 MySQL 網路監看與諮詢服務，是一套動態傳送的訂購服務，MySQL Network 會持續地監測用戶的資料庫系統，在潛在問題發生前先行警告用戶並協助解決問題，其中包含資料庫安全、效能調校、資料庫規格設計等領域超過 65 個專家諮詢規則。 對於 Mickos 宣稱 MySQL 已能處理企業關鍵需求如可靠性、可用性與延展性與易於部署等，Gartner 的 Graham 則認為 MySQL 等開放源碼廠商距離追上 IBM、Oracle 等企業級資料庫還有很長的路。她表示，雖然開放源碼在企業資料庫市場佔有率會在 2010 年時由目前不到 1% 成長到 5%，然而對於市場領導者的衝擊仍低。IBM、Oracle 等私有資料庫商也會持續透過提供免費版本與釋出程式碼等方式，阻礙開放源碼對手的追趕。 對此 Mickos 有他自己的看法，他認為他們視用戶為學習對象而非資料庫舊有的競爭對手。相較之下他們做法有很好的成果，不然的話，YouTube、Google、 Yahoo 與 Cox Communications、NASA 等使用 MySQL 的組織也不會有那麼好的表現。 MySQL Enterprise 支援 11 種平台，包括紅帽 Enterprise Linux、SUSE Linux Enterprise 伺服器、微軟 Windows、Sun Solaris、Macintosh OS X 與 HP-UX。價格採取年度訂購模式，每部資料庫伺服器從 595 到 4995 美元不等，MySQL Enterprise 伺服器軟體與技術支援已經可供用戶訂購，MySQL Network 監測與諮詢服務預計於本季稍晚推出。
