推薦一本 Python 工具書

Created at Friday, 25 April 2008 08:00      Last Updated on Tuesday, 11 January 2011 17:44 Written by 李貞臻
電子報接下來要跟大家介紹一本不錯的 Python 工具書－『 How to think like a computer scientist: Learning with python』第二版，目前正由 OSSF 同仁著手進行內容的翻譯工作，它是一本適合 Python 初學者閱讀的入門書，亦是 Open Book Project 一系列的電子書之一。這系列的書不只是 Freely Distributable ，還是 GNU Free Documentation License，代表您可以免費閱讀、散布和修改該書。我們將中文書名譯為－『如何像電腦科學家一樣思考』，中文內容將陸續分享在電子報的『技術專欄』中，本期首先登場的是『前言』與『序言』，希望給有興趣的朋友帶來實質的幫助。
