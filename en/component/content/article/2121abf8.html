BBC 製作開放源碼紀錄片系列

Created at Thursday, 16 July 2009 08:00      Written by 謝良奇/編譯
BBC 正在為旗下的 BBC Two 頻道製作包含 4 部份的一系列紀錄片，記錄 Web 如何且持續地改變人們的生活。BBC 日前宣佈 BBC Two 廣播網路與 World Wide Web 發明者 Tim Berners-Lee 進行合作，製作名為數位革命 (Digital Revolution) 的開放協同合作紀錄片。該紀錄片在製作流程上，將採取開放源碼模式，以記錄 Web 發明 20 週年。

此專案目的在於透過聽取線上使用者的意見與敘述，加上分享製作團隊的想法與概念，儘可能開放製作流程。根據 BBC 表示，該系列紀錄片的製作團隊透過在線上公佈紀錄片毛片，並分享主要論點、邀請來自 Web 社群的評論、敘述等，給予 Web 使用者早期使用製作內容的機會。

BBC Knowledge 委製主管 George Entwistle 表示，經過 20 年騷動的創新，是時候對社會在 Web 誕生後歷經的深切改變做出判斷。Entwistle 很高興 BBC 的閱聽人有機會在此專案製作中，扮演重要角色，他期盼在電視播映前，能看見此一專案在線上呈現。

數位革命是由新聞工作者與學者 Aleks Krotoski，在 bbc.co.uk/digitalrevolution 上以一系列 BBC 稱之為部落格宣言 (manifesto blog posts) 所啟動。她邀請 Web 社群藉由分享意見與提供記事方式參與其中。此外，BBC 表示，該專案會引導使用者，進行一系列諮詢網路科學研究創制機構 (Web Science Research Initiative) 後所發展的線上實驗。這一場線上討論將形塑此一 BBC Two 紀錄片系列的製作，Krotoski 也將揭開許多特別的趣味報導，說明 Web 在今天是如何被使用與濫用，並且找出線索以評估 Web 以及人們不明確的未來。

BBC 與 Open University 共同製作的數位革命，將於明年播映。該系列紀錄片執行製片 Dominic Crossley-Holland 表示，這是相當重要且切時的系列紀錄片，Web 之父 Tim Berners-Lee 的加入也非常令人興奮。該製作團隊承諾將採取儘可能的開放方式，這個方式可能將對電視未來製作的方式，有著深遠的影響。
