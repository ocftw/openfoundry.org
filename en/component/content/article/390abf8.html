Novell 財報顯示 Linux 業績成長 SUSE Linux 獲選 LinuxWorld 最佳企業伺服器

Created at Tuesday, 29 November 2005 08:00      Last Updated on Friday, 09 March 2007 00:45 Written by 謝良奇
Novell 發表第四季財報及年度財報，其 Linux 營收較去年第四季成長 418％，為六千一百萬美元，顯示 Novell 的 Linux 經營有不錯的進展。日前 Novell 的 SUSE Linux Enterprise Server 更獲選為 LinuxWorld 的最佳企業伺服器散佈套件。

第四季財報 Novell 營收為三億兩千萬美元，去年同季則為三億零一 百萬美元。2005 會計年度 Novell 營收為十一億九千八百萬美元， 稀釋後普通股股東純利為三億七千八百萬，或普通股每股稀釋後 86 美分。其中包含與微軟間的法律協議四億四千八百萬。 2005 會計年度 Novell 的 Linux 營收為一億四千八百萬，較去年成長 278％。第四季 Novell 的 Linux 平台營收包括來自 Open Enterprise Server 的四千六百萬與其他 Linux 產品及服務的一千五百萬。 Novell 旗艦 Linux 產品 SUSE Linux Enterprise Server 的第四季 銷售量，扣除企業級授權採購後為 6 萬 5 千個單位。 Novell 的 SUSE Linux Enterprise Server 日前更於德國法闌克福 的 LinuxWorld 會議上以 53% 的票數擊敗 Red Hat 的 Enterprise Linux、Mandriva，獲得第六屆 Linux New Media 的最佳企業 Linux 獎項。 Novell 執行副總裁兼技術長 Jeff Jaffe 表示，該獎項肯定了 Novell 在創新與品質上的努力，他們盡力讓 SUSE Linux Enterprise Server 符合企業運算需要，該獎項要與和他們一同努力的開放源碼社群及用戶分享。 盡管數字顯示 Novell 的 Linux 業務漸好，其 NetWare 業務也有 3％的小額上升。Novell 董事長兼任執行長 Jack Messman 表示， NetWare 將逐年衰退。另一項持續成長的業務則是 Novell 的辨識驅 動技術（Identity-driven computing ）。Messman 說，他們已準備 跨入 2006 會計年度，抓住其成長中的 Linux 與辨識領域的機會。
